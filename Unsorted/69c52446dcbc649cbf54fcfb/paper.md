# 摘要

侯孝贤作为台湾电影的代表人物之一，因其一贯的关怀主义和“苍凉"的风格，被认为是当代中国乃至国际最重要的电影人之一。本文主要从历史、景观、人物三个方面综论侯孝贤的电影，进行较为全面的研究，全文除绪论、结语外，主体部分由三章构成。

第一章论说侯孝贤电影的景观呈现，从电影美学与叙事出发，包括长镜头的普遍运用、自然与人文的交叠、写实与抒情的融合等内容，指出侯孝贤透过声音和影像的辩证，以庶民记忆来质疑历史大叙事。在空间处理上，不但以长镜头深焦呈现室内多层次空间的活动来反映家族和历史命运纠结的兴衰荣枯，更以自然景观和人文景观交叠出丰富的空间层次，来铺陈多重殖民的生活经验。

第二章论侯孝贤电影的人物描摹，从自传体裁与边缘人物出发，包括个人自传、戏偶代表、女性角色等内容，记录到1980-90年代侯孝贤的所有作品主体都很清楚地以自传体呈现，并且特别钟爱于个人的人生轨迹描写及边缘人物作为题材创作，透过回忆唤起历史分期、年代称号、朝代及描述时间架构的问题。而在检视、探索过去时，以宽美为代表的女性角色在他的电影中提供了另一种历史时间的形态，来讲述非指导性的个人历史。

第三章论侯孝贤电影的历史书写，以台湾新电影为开端，在整个新电影运动时期持续聚焦历史，张扬表现本土意识，通过来自本土作家沈从文的影响和来自台湾一代影人集体创作的推涌两大原因，述说包括缘何聚焦历史、日常琐事记录、重大事件还原等内容，并通过四部作品《尼罗河女儿》《恋恋风尘》《好男好女》和《悲情城市》为例，展现侯孝贤对历史与人性进行非戏剧化的真实书写。

侯孝贤在历史、景观与人物表述中讲述历史大叙事，真切地关注着在族群利益纠葛中善良的人民与他们质朴的生活感情和梦想，挥洒出一份沉淀浑厚的历史掌握与关怀。

关键词：侯孝贤，台湾，长镜头，自传

# 绪论

# 一、研究背景

在20世纪80年代中国台湾新电影运动时期，侯孝贤就因其电影艺术成就的高超和文化蕴含的丰富而备受国际瞩目，更因《悲情城市》于1989年荣获威尼斯影展金狮奖使得他至今都是国际影坛最有影响的导演之一。

侯孝贤（HouHsiao-hsien），1947年4月8日出生于广东梅州，1948年移居台湾，电影导演、监制及编剧。截至2018年的导演生涯中，侯孝贤获得戛纳电影节最佳导演奖、威尼斯电影节金狮奖、洛迦诺国际电影节终身成就奖、亚洲电影大奖终身成就奖等国际电影大奖，国内也曾两度获得台湾电影金马奖最佳导演奖和中国长春电影节最佳导演奖。

侯孝贤是台湾电影的代表人物之一，是台湾新电影运动的领军人，他至今共导演19部电影长片。侯孝贤执导的19部电影长片分别为：1980年《就是溜溜的她》、1981年《风儿踢踏踩》、1982年《在那河边青草青》、1983年《风柜来的人》《儿子的大玩偶》、1984年《冬冬的假期》、1985年《童年往事》、1986年《恋恋风尘》、1987年《尼罗河女儿》、1989年《悲情城市》、1993年《戏梦人生》、1995年《好男好女》、1996年《南国再见，南国》、1998年《海上花》、2000 年《千禧曼波》、2003年《咖啡时光》、2005年《最好的时光》、2007年《红气球的旅行》、2015年《刺客聂隐娘》。

侯孝贤作为台湾电影的代表人物之一，倡导艺术重视人文关怀，将视野聚焦于人本身，顾因其一贯的关怀主义和"苍凉"的风格，被认为是当代中国乃至国际最重要的电影人之一。同时侯孝贤的叙事混杂了东方式的内省和凝练，以及西方式的客观与疏离①，承载着台湾百年以来的社会变迁，这段历史在他近四十年的电影创作中一一展开，而他至今所导演的19部影片，以艺术的形式记录了台湾上下五十余年来的历史与发展。

侯孝贤是台湾80年代新电影运动的领军人物，是战后出生的大陆移民的孩子，随着六七十年代台湾经济起飞、政治革新，尚未成人便随着父母迁台。侯孝贤长期生活在台北南部的小城凤山，曾在台湾本土以及电影界经历了许多风雨。他与杨德昌代表了台湾新电影运动的内部的不同艺术取向的总体风格，他们犹如新电影的黑白两道，相辅相成，构成了台湾新电影运动令人叹为观止的景观。不论对侯孝贤或对台湾电影来说，他的电影作品树立了一个里程碑，是台湾电影文化改变的结合，是创意合作的持续努力。侯孝贤和他的电影并非只是个人成就，而是可被视为当代台湾电影的缩影，从票房导演到国际作者导演，从票房毒药到台湾电影教父，侯孝贤与电影本身共同行走在一条迁回道路上，从兴盛到受挫，再把他推向家乡领土之外。

从很多方面来说，侯孝贤是1990年代台湾电影的同义词。无论海内外，同辈导演中属侯孝贤最受到瞩目和检视。①侯孝贤以沉思、远距的摄影，时间上的省略，以及隐晦的因果关系构成他的长镜头美学，再加上他对人世充满怜悯的观察，常被视为是东方超然的表现，因此侯孝贤也被定位为东方的顶尖导演。相较于国际上的一致赞誉，侯孝贤在自己的土地上则被视为一触即发的火种，快速燃烧着中国台湾多元的文化景象。1980年代中期，在短暂但关键的电影研究中，侯孝贤解读为当代文化政治的表征。他精简、感性，甚至禁欲般的写实主义风格，唤起了批评界空前的思索。对于台湾电影该走向艺术或商业的争辩，为一向闭锁及充满派系斗争的台湾电影评论带来了一道强光。吴念真曾经这样说过侯孝贤：“在台湾，我对电影和人都喜欢的就只有他，并且我十分敬重他，不仅在于他日常的为人处事，总是很亲近，更在于他对电影表现出的大气魄，我发自内心的敬重他。”③

# 二、研究综述

# 1.侯孝贤新电影时期作品的分析

侯孝贤从1980年开始执导他的第一部电影长片，在台湾新电影时期，侯孝贤一共执导了10部电影长片，学界对此有专门的研究。陈晴指出，台湾新电影始于20世纪80年代，受到法国新浪潮和左岸派的影响，有别于前期的浪漫家庭剧，侯孝贤的创作开始回归现实人生与真实时代①。刘立行和沈文颖在合著的《电影文化与产业》有一章节专门对台湾新电影运动进行研究，其中写道“台湾新电影最主要的贡献是让台湾极具才华的一批电影人找到了抒写的空间，在各自的叙事架构中，描绘台湾迷人而多样的面貌....提起台湾新电影，一定会提到侯孝贤这位最具代表性人物。由于很少导演在一部原创的电影中，以逼真如纪录片般，把个人的人生或生命历程与大时代的命运交融在一起。"②李劫刚在《论侯孝贤电影的艺术品格》中写道“侯孝贤作为台湾新电影运动中值得敬重的导演之一，他有别于台湾传统电影的叙事模式和技巧，其电影体现了对社会人生真诚的关注，抒发着面对个人成长与人世沧桑的感慨。"③

# 2.侯孝贤电影形式与风格的阐释

侯孝贤的电影风格在一众中国导演乃至世界导演中独树一帜，国内外对他在形式与风格上的研究有许多。自新电影运动起，至将自己的人生观和艺术观锤炼之后，侯的电影形成了东方式的感性把握世界，以崇尚自然、含蓄达观的态度形成了诗意的影像风格。侯孝贤以沉思、远距的摄影，时间上的省略，以及隐晦的因果关系构成他的长镜头美学，并加之对人世充满怜悯的观察，常被视为是东方超然的表现。④袁玉琴曾在《世界华文文学论坛》中评价到，“侯孝贤的影片，融通中外文化，以特有的风情格调与美学特征自成一家。在长镜头纪实性镜头的简略勾画中，着力营造银幕视象独特的艺术意境。"蔡潇在《再植存在—论侯孝贤电影的日常场景建构及其精神渊源》中也论述到侯孝贤与西方写实主义影像十分相像，并传承安德烈·巴赞的电影理论，但其美学上的电影艺术却有别于西方的写实主义。侯孝贤在找寻属于自己的电影风格过程中结合了台湾本土的抒情方式和中国传统美学，并找到了应当呈现的叙事态度与传统哲学的回归，侯孝贤电影的形式和风格都有其自身的侯氏表达。

# 3.侯孝贤电影中景观场景的论析

在探讨侯孝贤电影中的景观研究时，主要是从视听语言、画面及场景来进行，张霭珠在《全球化时空、身体、记忆》中花了大量篇幅分别讨论侯孝贤电影中声音和影像的辩证、自然景观和人文景观交叠的多重殖民空间、室内空间的多层次纵深、倒叙套层时间与意义位移这些主题，加以对景观进行多层次研究。晏凌在《叙事与景观的平衡—看侯孝贤的 $<$ 冬冬的假期 $>$ 》中分析侯孝贤的电影《冬冬的假期》写道，“作为侯孝贤最初确立自己风格的作品之一，在叙事上与其前期和之后的作品都有区别，它较好地维持了景观与叙事的平衡.....在景观铺陈中渐渐累积出故事。”①

# 4.侯孝贤电影人物形象的研究

侯孝贤电影中呈现的人物大多源自现实生活，除他钟爱的自传体式电影叙事，随着时间的推移和导演的生活经验积累，侯孝贤将目光逐渐更多地聚焦于边缘人物。张琼在《论侯孝贤电影中的边缘人形象》中说道，“他 (这里指侯孝贤)把边缘性看作生命的本真状态，看作生活在不同时代和社会形态中的人的共同特征”，“侯孝贤电影通过去除先在观念的影响、重塑边缘人形象体现出对既定认同框架的质疑和解构。"②侯孝贤的电影创作在20世纪90年代中期发生明显转变，周仲谋在《空间拓展·时间跨越·性别转向—论侯孝贤电影创作的转变》中提出,侯孝贤有意识地在银幕上呈现边缘女性，甚至置于主角的位置，展现一种非常规式的生活状态，不仅道尽作为边缘女性自我执着的独立人格，更将其孤独意识与生命体验呈现出来，③表明侯孝贤电影中的人物指引台湾现实格局，引发性别生命思考。

# 三、研究意义

对侯孝贤的电影研究，不仅是为了深入了解一个导演的创作过程，还具有如下几个方面的意义。

首先，通过研究侯孝贤电影的历史，可以对1980年代中国台湾新电影运动的起因和过程有更深入且正确的艺术反思。1982年8月28日至1987年1月14日，即台湾新电影运动的首部电影《光阴的故事》与《儿子的大玩偶》①公映至50 多位台湾电影界名人发表《台湾电影宣言》，是台湾新电影运动掀起的时期。台湾新电影运动几乎是与香港新浪潮电影（以1978年严浩的《茄哩啡》公映为标志）和中国大陆的新电影运动（以1983年《一个与八个》的问世为标志）同时兴起的，他们都是属于世界新电影运动的一部分。但是，台湾新电影运动是在以琼瑶文艺爱情片为代表的台湾商业片、类型片创作热潮之后应运而生的，是与好莱坞式的"梦幻工厂"与明星制批量生产的台湾主流商业电影划清界限的一次集体尝试，反应台湾“尚未开发国家到已开发国家的阶段中的高等游民”，即台湾青年白领阶层的艺术趣味上在台湾电影观众圈位置的重新定位，也是台湾电影第一次高度自觉地有意识地在世界电影总体格局中去建立严肃的人文电影新传统的一次迅猛的前卫的实验电影尝试。从而，彻底扭转了台湾电影发展方向。

其次，研究侯孝贤的电影，可以在权衡商业与艺术电影两种力量上进行更深层次的思索。80年代因为“新电影”出现，得以培育出世界级导演，使国际影坛知道中国大陆、中国香港电影外，还聚焦到中国台湾电影。1989年侯孝贤的影片《悲情城市》得到威尼斯金狮奖后，不但使华人电影扬眉吐气，亦使观众正视"新电影"威力，引发商业片与艺术片之论，摆在台面上抗衡。由于侯孝贤的灵感较多源自其原文化泉源即生活经验，而源自电影内部思索较少，所以在与其他优秀电影互相交流的同时，更为中国电影，尤其是侯孝贤的电影，提供了另一条道路。“新电影”远离观众市场，更突显出电影产业的的弱点，电影潮流通常以五年为期，就得更换题材内容，“新电影"面临同样周期，走入颓势。只是这回众人拾柴火焰高的"新电影”开始展现出台湾本土魅力，真正迎向国际主流。然而台湾电影常年积累的弊病在当下全涌现出来，包括政府政策、社会环境、传媒结构等方面，使其难逃衰竭命运。台湾电影产业难有健全工业结构，无法像香港电影工厂生产商业类型片，或者像中国大陆拥有产业网络链，以人口量为后盾。台湾电影的优势在于汉文化承袭的人文素养，唯有善用有利资源，才能创造出电影条件。尽管台湾电影有人才紧缺、资金不足、外片占据、市场过小、创作自溺、忽视观众等问题妨碍发展，但知己知彼，方能克服困难，创造贏面，②研究侯孝贤电影创作为

商业和艺术电影的权衡提供了条件。

最后，研究侯孝贤的电影，可以有助于我们对电影所有的基本元素与电影艺术的创新进行反思。在侯孝贤的作品中可以看到，电影的一些基本元素并未在其中展现，更甚的是也不同于过去传承的小说或戏剧艺术传统(特别是不同于好莱坞演艺事业对加速，割裂的赚钱崇拜），这使他似乎置身于现代电影的问题之心。不论是用巴赞所倡言的"真实的无缝天衣”，或是用德勒兹所钟爱的“影像—时间”做论题，侯孝贤的场面调度都能用优越的方式实现他的现代电影思想。不过，他的电影所具有的现代性并非经由演变而来，也并非是由其它电影经验制作的电影，侯孝贤的电影是以历史、本土文化、个人经验、地理意识制作的电影。

# 四、研究思路及方法

本文从景观、人物、历史等三个维度，来综合研究和整体审视侯孝贤电影，对其电影进行细致分析。

第一，通过侯孝贤电影的景观呈现分析，引出台湾的历史建构与关照，指出侯孝贤透过声音和影像的辩证，以庶民记忆来质疑历史大叙事。在空间处理上，不但以长镜头深焦呈现室内多层次空间的活动来反映家族和历史命运纠结的兴衰荣枯，更以自然景观和人文景观交叠出丰富的空间层次，来铺陈多重殖民的生活经验。

第二，通过侯孝贤电影的人物描摹分析，从自传体裁与边缘人物出发，记录1980-90年代侯孝贤所有的作品皆以自传体形式呈现，并且特别钟爱于个人的人生轨迹描写及边缘人物作为题材创作，透过回忆唤起历史分期、年代称号、朝代及描述时间架构的问题。而在检视、探索过去时，以宽美为代表的女性角色在他的电影中提供了另一种历史时间的形态，来讲述非指导性的个人历史。

第三，通过侯孝贤电影的历史书写分析，以台湾新电影为开端，以四部作品《尼罗河女儿》《恋恋风尘》《好男好女》和《悲情城市》为例，展现侯孝贤电影在中国电影史乃至国际电影史的地位及意义。

侯孝贤电影的研究，需要良好的理论功底，娴熟运用文本阐释学、接受美学、文献学等层面的方法进行整体上的把握和切入。

# 第一章景观的呈现

居伊·德波在《景观社会》中谈到，为以景观的方式呈现自身，需要无限堆积生活的方方面面，整个社会生活在一定时间内积聚景观后，所有过去真实存在的、历历在目的社会经验已经全部转化为再现①。影像中的景观大多来自日常经验，借助导演自身的艺术拍摄方式进行特殊化呈现。侯孝贤电影的景观呈现，包括电影美学、诗学、叙事与镜头皆是重中之重，这里所探讨的是长镜头、自然与人物景观的交叠以及写实与抒情融合中的景观呈现。

侯孝贤的作品是20世纪八九十年代台湾电影研究领域中最被中外学界讨论的对象。在观看侯孝贤电影时，不论是他早期的贺岁片、或被归类为新电影的作品、或《悲情城市》后的片子，都看到他呈现景观的主要方式是不断地与电影语言对话。比如人们常提及的侯孝贤镜头的长短，他在《儿子的大玩偶》后的风格的确开始向一个模式推进，这个模式便是 (到《戏梦人生》为止）有着减少剪接、逐渐增加镜头长度的倾向，电影中的各种技巧也越来越简约或淡化：放弃了叠印、溶接、交叉剪接；没有了大特写及推拉镜头；几乎不用淡入淡出；很少观点镜头及正反拍。剩下的仅是切接、跟拍、摇移、中特写及大远景，加上他偏爱纪实性镜头，所有要建立的景观与叙事并非以观众（局外人）的角度呈现，而是通过角色的眼睛即角色的主观性视角体验影片中的人生与世界。这种简约风格的形成存在少许客观因素，但更多代表了侯孝贤个人强烈的偏好因素。再谈到侯孝贤从1980 年的第一部《就是溜溜的她》到1998年的《海上花》，他对电影诗学的掌握越来越成熟洗练。此处所谓的诗学指景色诗学，是对自然景物的植入，遵循道家“物我同一”、“天人合一"的哲学观②，是自觉产生的，根据经验无限创造出的一系列美学语言和电影语言。而这种艺术创作表达逐渐构筑成固有的侯式电影形式。

侯孝贤电影中对于景观的呈现透过声音和影像的辩证，以庶民记忆来质疑历史大叙事。在空间处理不但以长镜头深焦呈现室内多层次空间的活动来反映家族和历史命运纠结的兴衰荣枯，更以自然景观和人文景观交叠出丰富的空间层次，

铺陈多重殖民的生活经验。

# 第一节 长镜头的普遍运用

长镜头的普遍运用，本土景色诗学的镜头表达，视觉和听觉的象征与结合，个人空间的景框层次，历史时间的物化表现，后期跳跃式剪辑的意义，地方语言及配乐歌曲的选用，重复构图背后的历史喻意，刻意简化剧情留足空间想象的处理，对抗好莱坞美学的段落化结构，这些都是侯孝贤电影至今超越绝大部分台湾电影的形式技巧。其中以长镜头为主，作为电影的景观呈现的形式之一，在侯孝贤的导演作品中占据了不可替代的地位。

法国导演居伊·德波曾这样描述景观，景观是具有可观看性的，是一种幻象，它的建构应当是一种寄托于感性的表象的存在，从而在各类复杂的影像中表现出不同形式①。电影之于景观的呈现需要透过影像，历史的真实记忆往往从影像中表现，以此于景观之中建构属于导演仅有的记忆。

# 一、长镜头的坚持

对于侯孝贤一生中经历的时代变迁，为更贴近写实客观，他坚持用长镜头力持冷眼静观，以呈现一种生活景观。为尊重表演者，侯孝贤拉远了镜头与演者的距离，摒除了夸张的特写、表情与动作，电影的戏剧性完全诉诸自然法则下的真情流露，任由情感淡淡渗出银幕，直接呈献给观众。对于最初使用长镜头拍摄电影的原因，侯孝贤曾经说，“我既没有看过巴赞的书籍，小津安二郎的影片也是拍完《童年往事》后才看到的，之所以常用长镜头，一是当初电影器材的缺失，二是起用了非职业演员的需要，让他们能够表达出感情，发现一个镜头下来有魅力，比再去剪辑还要有魅力，慢慢就喜欢用长镜头。三是拍摄时选配音后剪辑，不用长镜头怕剪不准”。此后长镜头逐渐成为侯孝贤基本镜头语言的代表形式,侯孝贤常用后景、远景来表现景观中的重要细节，或者通过画外空间表现。

所以长镜头成为侯孝贤拍摄电影时的代表方式是经过一个漫长过程的，起初侯孝贤选择使用长镜头不仅因为当时台湾电影界并非发展出众多职业演员，资金的短缺也造成拍摄场景、器材、道具、人力都不足以支撑一部影片的完成，当然他的御用剪辑师廖庆松则对这样的说法不予认同，指侯孝贤“过于谦虚”。事实上，当时能将长镜头技术代入电影中是十分困难的，因为一个镜头的持续时间过长，极易导致沉闷而冗长，这种技巧需要导演具有深厚的文学素养与情绪拓展。在侯孝贤的长镜头里，镜头一般是不怎么运动的，所以就不能使用镜头的运动来构成画面节拍，而一般是采用构图和比较不容易注意到的细节信息来保持视觉注意力。

# 二、远距长镜头的呈现

侯孝贤常常将他纪录片式的摄影与沉默地旁观者融合在一起，以第三人的视角形成远距离观看，贯穿导演一如既往的长镜头表现形式，将远距离拍摄与长镜头相结合，建立起一种远距长镜头美学的风格标记。

侯孝贤电影中的景观呈现，不仅通过男女主角视角，旁观者往往也是主角，他们观看但不加以评论。这些旁观者经由"看见”，不加评论的让画外人触动同理心与敏感性。在《风柜来的人》《冬冬的假期》《童年往事》和《恋恋风尘》等电影作品中，侯孝贤大量采用远景长镜头观看他的乡里主体景观，拍摄如画般的乡村场景、农村的沉闷与在台湾小镇随处可见的作息。在陈坤厚掌镜的《风柜来的人》一开场，便展现了导演对于捕捉乡里族群景观韵律的着迷与渴望：在一个简陋的撞球间正在进行一场比赛，一位满脸皱纹的老人拿着粉笔在破旧的黑板上计分。这个场景将焦点直接放在老人身上，此时侯孝贤与陈坤厚感兴趣的不在于叙说故事，而是曝光易感光的底片，即在不干扰的前提下找出突出的脸孔与场景，以视觉人类学角度出发，从而达到呈现人物与环境关系这一特殊景观的目的。而早在《冬冬的假期》开拍时侯孝贤就曾表示：“西方电影启发我们可以脱离逻辑的压迫和蒙太奇规则的想法，而且可以摆脱无用的镜头”。①同样在《风儿踢踏踩》中，侯孝贤也以有着上镜特制的近海渔村做底，其中绝非是为了通过环境突出人物，或者通过叙事突出环境，侯仅是以你所看到的景观来呈现事物本质，并非为建立电影要素中的某种联系。

侯孝贤电影中的景观呈现经常透过远距长镜头加以表现。远距长镜头的摄影方式意味着本是沉默旁观者表露的长距离心理距离，给予一个可以呼吸与沉思的空间，迁回的叙事方式则代表着敏感旁观者的保留或对自我的压抑，因此呈现的景观必然冷眼且直接。影片《风柜来的人》里，浮躁的阿清跑到城市后，以大城市虚假繁华的景观呈现为界，转为静默的旁观者，所以当阿清爱上了租房隔壁的女孩，他最终选择一言不发。其中看上去很土的乡里膰腆反而成了优点，因为它让旁观者可以不涉于其中，以致破坏那原来的状态，那个“自然的法则”。

在电影《悲情城市》上映前与侯孝贤的访问中，他曾表示长镜头就是他喜欢的拍摄形式，这种拍摄美学与侯孝贤自身个性十分契合，对于自己平常看事情无法深入看到事件本质的性格，当无法运用主观思维带领观众进入电影世界时，更好的方式就是借鉴沈从文冷眼看生死的态度，在用客观角度拍摄时不免就会产生最大的宽容与深沉的悲伤。①事实证明侯孝贤是对的，《悲情城市》中对长镜头的普遍运用并未使画面流于单调疲乏，他在一个镜头内呈现多重画面焦点，并让大量人物不断出入镜头营造历史氛围，重新建构空间层次，最后用旁白 (画外音)与画面内部对话做对比，添上浓墨重彩的一笔。如其中在医院室内室外构成的两种情况和“小上海"酒家里不同房间交织的不同构图和层次，都赋予长镜头更多的景观意义。运用长镜头诉说《悲情城市》，二二八事件化身为一种艺术电影形式，使得艺术、人文、历史和政治间产生更多对话。

影片《戏梦人生》中的长镜头在展现戏剧景观时同样承袭了这一美学观，以岛崎君的回忆、与酒醉日本兵打架等事件叙事，远距长镜头让人物面貌几乎难以辨认；影片中八次出现李天禄的声音，对我们即将看到、可能看到但没留意、或是根本看不到的事情加以评论，旁白过后，李天禄六次现身镜头前。这些段落呈现的人生仿佛无可避免，天意秩序也在"戏—梦—人生"这样的时序安排出现，以超过七分钟的一镜到底长镜头展现，只以极少的镜头移动，重新框取作为停顿，就在这个单一镜头中，“戏—梦—人生"逐渐揭开。根据朱天文的说法，侯孝贤为了在历史的进程中实现对李天禄人生的构想，他为《戏梦人生》发展了一种新的剪接形式，称之为“气韻剪接法”。侯孝贤想做的是模拟云朵行进的叠盖与积聚，生成厚而重的天空；同样地，漂流于戏剧、梦境、人生漂流的缕束也协同创造了满载着"神秘的原因"与不可理解动机的历史。 ①

侯孝贤的电影风格不仅影响台湾的电影美学，也影响了亚洲的电影导演。詹姆士乌登（JamesUdden）指出，侯孝贤的固定镜位和长拍②影响了杨德昌、蔡明亮③：杨德昌1980年代的城市三部曲平均每个镜头不超过15秒，而到1990年代的系列电影则每个镜头平均在25秒到50秒之间；蔡明亮的电影则从《青少年哪吒》(1992）每镜不超过19秒到《你那边几点？》 (2001）每镜都超过1分钟;而蔡明亮《你那边几点？》与《河流》室内打光的明暗对比和局部光晕也有侯孝贤的氛围。其他受侯式风格影响的亚洲导演还包括韩国HongSang-soo、LeeKwang-mo，以及日本导演HirokazuKore-eda等人。④侯孝贤的导演作品大多是使用这样的大量长镜头呈现远距景观，挖掘台湾民间悠长的生命力，以客观写实角度提供给观众一个认识理解台湾历史轨迹的机会。

# 第二节自然与人文的交叠

侯孝贤的电影在景观呈现上不但以长镜头深焦呈现室内多层次空间的活动来反映家族和历史命运的兴衰荣枯，更以自然景观和人文景观交叠出丰富的空间层次，来铺陈多重殖民的生活经验。自然景观的呈现需要运用导演独有的艺术表达、场面调度和画面元素给影视创作带来足够的审美空间，人文景观大多以碎片化呈现，通过营造一个生活空间呈现导演渴望展现的现实生活的某个侧面，侯孝贤在影片中以自然景观与人文景观交叠出现，借助画面连接导演与观众的情绪交流，引发景观背后思考下的情感共鸣。

电影《悲情城市》是侯孝贤作为导演的代表作品，以一个家族的兴衰折射台湾二二八事件背景下的历史变迁，侯孝贤凭借此片获得威尼斯国际电影节金狮奖和威尼斯国际电影节西阿克特别奖。这里以《悲情城市》为例，对其中自然与人文的交叠剖析景观的呈现。

在交错的江湖情和历史情中，以《悲情城市》为代表，除了以听觉和视觉的矛盾来凸显二二八事件官方论点和人民记忆的落差，更以自然景观与人文景观的交叠来丰富空间层次，形成侧重点，铺陈台湾民众多重历史时空的生活经验。《悲情城市》透过远距的自然景观和多层次纵深的房屋内部空间（林氏家宅、小上海酒家、医院等）交映，人物从镜头两侧或后侧出入于室内外空间，以及空间影像和声音意象的相互汇入，来召唤那种已存在的空间记忆和生活经验，也描绘不同殖民者进入在地空间后引起在地文化形式和内容的偏移，以及偏移过程中的冲突、协商、承接、转化，成为在地文化的一部分。①

# 一、自然景观的聚焦

《悲情城市》以空镜远景呈现的自然景观一共有13个②，兼具衔接场景、叙事、转换时空和象征隐喻的作用，蕴含了侯孝贤受沈从文影响③下的保持距离、冷静、不失悲悯的态度，凝视着发生在这片土地的一切。

在自然景观中，最具代表性的是九份山崖俯瞰海湾渔港的场景。这个画面多次重复出现，配以不同的声音或旁白，一方面以大自然照常运转的四季时序对照个人与家族生育、成长、节日、婚庆、丧礼等日常生活节奏，另一方面也铭刻历史变迁的轨迹，汇集殖民权力、走私贩毒、家国存亡等多重意义。④《悲情城市》所展现的九份人文景观也因历史随之消散，塑造着九份的文化想像与家国怀旧。

伴随日本战败，宽美在文清的陪同下前往山上的医院工作，画面融入宽美的画外音，讲述着她的日记：想到日后能够每天看到这么美的景色，心里有一种幸福的感觉。台湾光复，宽美期待着重回安定与平和。然而这种期待随着陈仪代表国民党政府接收台湾后而落空，人民面临的是动荡时局与现实生活物资的匮乏。在林老师、宽荣、文清、何记者等一群知识分子忧心时局下，共同唱着《流亡三部曲》，这时画面跳转到海上陆地的远景，配之自然景观的音乐也由片头的日本音乐改成国语歌，反映政权的转移。这群青年唱着“哪年、哪月才能回到我可爱的故乡"以解思乡之情。之后的画面同样是山川远景，思乡的歌声转为隆隆的雷声，把时空转移到雷声大作的午后，宽美在医院收纳床单，隔窗病房内是上海返乡的文良由于战争带来的创伤而变得疯癫，双手被铐在病床上流泪。上述画面由九份的山海景观伴随着思乡的歌声和隆隆的雷声，套之知识分子怀念祖国，宽美处理日常琐事以及文良回到祖国后的伤痛，预示着即将到来的变局。①随即剧情发展，自然景观不断被用来对比反讽政治暴力和历史悲情。在陈仪作有关二二八事件的广播时镜头带向空镜景观，镇压人民的讯息随着九份满山的芒花而扩散开来，事件结束后宽荣躲入深山并娶妻耕读，画面随着文清到山区看见美丽的梯田和水牛犁田的景观，但这般景观过后便是警察潜入深山逮捕宽荣的镜头。

# 二、人文景观的捕捉

“日常生活的领域—现实一切其它形式的母体，它改变得很慢，战争、瘟疫、地震和革命都摧毁不了它。电影即倾向于探索日常生活的这种构造。不仅帮助我们理解我们特定的物质环境，并且向各个方向扩展它。电影实际上是把整个世界变成了我们的家园”。②

《悲情城市》的人文景观包括室内（私人）景观和公共景观。侯孝贤在处理室内景观时多以固定长镜头、深焦，放在入门处或门外朝室内拍摄，人物一般从镜头两侧或房屋后方出入。这样拍摄一方面有物质条件限制的原因，但大部分是就电影美学而言，透过静止的景深长拍可以拍出室内景观的纵深，让观众透过交错叠加的门框、窗框来观看剧中人的出入活动行为，加之室内景观往往有前、中、后三层景深，各自情境共同进行，不同层次的影像互动让画面更有张力。③进一步看，藉由长拍景深的多层次室内景观呈现出人与空间互动情境的变化，映射个人或家族在时间轨迹中的变化。

日常场景在侯孝贤的电影中十分重要，不仅给予背景和情节一定的空间，还赋予其生命和美学意义。④《悲情城市》中林宅大厅的圆桌景观一共出现过三次，三次构图也相同，画面中仅是一家人共同吃饭的场景，却蕴含着三重不同的导演意图，点示了林家命运的转变。第一次圆桌构图是因林文雄得子和小上海酒家开张引入林家人围桌吃饭；第二次林家人围桌吃饭是因小上海酒家被迫停业，文良和文清从监狱获释回家，前景是文清和宽美分坐圆桌两侧，宽美剥豌豆备菜，中景透过窗框看到已经痴呆的文良，伴随文清对狱友血书送到妻儿手中的倒叙回忆切回林家人围桌吃饭的景观；第三次围桌吃饭的场景是在片尾，林文雄死于江湖暗杀，文清被带走，女人们准备碗筷，桌上只剩苍老的阿公、痴呆的文良和未成年的男孩。侯孝贤用长拍景深的室内情境呈现这出吃饭的人文景观，道出家族的命运和渴望回归祖国的悲悯心境。①

《悲情城市》的公共景观包括医院、酒家等，都以暗调封闭或半封闭的空间来呈现殖民权力或文化冲突。如宽美在医院写着日记，画外音听到“你哪里痛”、“肚子痛”、“头痛”，随后画面切到一群医护人员学习普通话，他们努力克服台语和日语口音，好迎接祖国来的病人。还有酒家中林文雄与上海帮老大会面，林文雄的台语需通过他人翻译成广东话再译成上海话等等，这类于公共景观发生的事件描绘了历史交替过渡空间的复杂化，也表明渴望迎接祖国与文化隔阂的摩擦。另外，最为代表性的人文景观呈现是监狱内的场景。在监狱这个暗调的封闭空间中，林文清和一群青年因涉二二八事件被捕，文清见证他人被枪决和悲情歌唱，对比之前歌唱祖国的《流亡三部曲》，表达了导演侯孝贤对当时回归祖国的欣喜和面对国民政府腐败专政的抑郁心境。②

景观属于社会的一部分，作为酝酿意识的工具，侯孝贤电影中的景观呈现看似通过自然或人文构成某一或多重重复场景，实则客观之下表达导演主观意识，但他不愿左右观者的心境与态度，一切观感源自写实画面，感悟源自观者自身，这就是侯孝贤处理景观呈现的功力。

# 第三节 写实与抒情的融合

新电影运动的推动者之一陈国富曾说过：“究其根底，我们的电影运动，其史诗和吾国历史相一致，乃是对国家身份认同的热烈的追寻”，侯孝贤电影从写实主义传统出发，以抒情写意为美学原则，一方面为表感性存在下的家国情怀，一方面致力冷眼观世界的美学拍摄方式，侯孝贤在调和写实与抒情下，形成了他抒情写实的电影风格。

# 一、客观写实的坚守

1983年侯孝贤与台湾新进女作家朱天文合作，为陈坤厚执导的新电影作品《小毕的故事》当编剧，这时《儿子的大玩偶》开始接触为中影网罗留学生杨德昌、柯一正、万仁、曾壮祥，看了不少他从外国带来的费里尼、黑泽明等现代电影，开始理性思考"电影怎么拍"的问题，为他的电影写实做下铺垫。此后又读了朱天文推荐的沈从文的自传小说，深受其中抒情特质的启发，开始结合自身艺术经验，参照西方现代电影作品的成果，重新认识电影，靠自己的独立思考去拍摄电影。

人在这个时代里面，在一个环境里面，就会变得跟动物行为一样，会有所谓的感情，都是源自于最自然最原始的那种东西，没有经过很多的装饰。①从1983年《风柜来的人》开始，侯孝贤一直坚守客观写实的拍摄方式，努力记录台湾某个阶段、某个状况下的生活层面，但又与此同时引出其个人主观情怀，寓客观于抒情中。《风柜来的人》记录了小镇青年在面对即将入伍当兵和繁华都市文化的格格不入；《恋恋风尘》在乡村传统与都市文化的对立下见证了台湾70年代初的社会转型，以及小人物面对大环境下的悲哀无措；《冬冬的假期》仍旧以小镇与都市生活作比，记录80年代存活的年青一代由童年变为青少年的成长过程；《童年往事》叙述导演自身，回忆了当时举家从大陆移民逃到台湾的悲剧，更是时间流转背后的一代又一代，不断凋零、成长、再凋零；《尼罗河女儿》探讨了80年代都市霓虹灯下的腐败问题,并真实呈现了当下青少年生活在这样背景中的无所事事与空洞虚无；《悲情城市》更溯源时间长河，直追台湾四十年来政治神话结构之症结，强调台湾40年代的历史大事件，如何由日占统治全面转换成国民党领导的过程；而他的作品《戏梦人生》，更将台湾史往前推，藉布袋戏大师李天禄的生平，自清末记录到日本人战败离开台湾为止。由这么清楚的作品时间脉络来看，侯孝贤不只是有自觉的艺术家，也是有使命感的历史家。

# 二、主观抒情的自觉

朱天文曾这样评价侯孝贤，“（侯孝贤）基本上是个抒情诗人，而不是说故事的人。"侯孝贤的电影特质非叙事和戏剧，而是主观抒情总会自然占据一半。“坏孩子”是侯孝贤带有自传性的表现角色，在他的身上折射出侯孝贤自身的成长经历，他是台湾社会现代化以后被竞争机制抛出的边缘人物，精神和实体上都是无根者，是都市的飘泊者。这是台湾电影作家中从未有过的角色类型，侯孝贤将这一角色类型引入台湾影坛的中心地位，但与此同时客观写实又占据另外一半，因为他镜头下的画面用纪实性风格和浓郁意识自传体角色将客观表现因素“天人合一”般的融合在一起。

侯孝贤在《风柜来的人》中以一种散文化的方式来拍摄影片，他真实表现生活事件而不讲述故事情节，冷静而不煽情，连演员都是初登银幕的陌生人。而到《恋恋风尘》时，侯孝贤常常只给观众看到事情的开始与结果，却并不给观众直接看到事件本身。比如阿云的移情别恋，在开头提了一下邮差，之后再见时她已经和邮差在一起了，侯孝贤似乎刻意回避了那些看起来可能会具有过度戏剧性的桥段，他不希望这些过分戏剧化冲突化的场景破坏了一种悠长的意境，让观众出戏，这其实类似于国画里"留白"的技巧。因此侯孝贤的影片常常具有反复观看的趣味，它所呈现的是一些叙事上比较静止的离散片段，但观众却可以依据前后把中间的情节与人物关系的变化补全，这种留白，应该也是影片具有水到渠成的强烈张力的重要原因。

《恋恋风尘》中最为重要的景观之一在片尾阿公与阿远共同种番薯，阿公说“照顾这些番薯比巴参还累，好不容易种的番薯，要精心照料，也要除藤，但台风一来，还是要减产”，这样客观写实的景观在影片中出现多次，虽然人们可以做一些什么努力,但在人生面前似乎总有一些不可抗拒的因素,让人的努力白费，即使费尽心力，人生仍然是很难如意的。幸好如今有独特的电影媒体与侯孝贤导演的用心，得以提供抒怀人生叹息的舞台。

再看侯孝贤的传世力作《悲情城市》中设计了一系列的叠印和字幕插入。如影片开头在收音机响起时画面出现展示历史真实性的文字：一九四五年八月十五日，日本天皇宣布无条件投降，台湾脱离日本统治五十一年；在影片末了呼应叠印：一九四九年十二月，大陆易守，国民政府迁台，定临时首都于台北；①更在整部作品中出现多段完全无主词的对话。

《悲情城市》是一种带有强烈主体意识与文化感的历史巨片体制，其中悲天悯人的冷眼俯视人间的艺术高度，那种历史与人性、还原性镜头与心理视角的融汇，那种悲壮雄放的编剧精神，展现了侯孝贤客观写实与主观抒情下的家国人生。这些文字以客观写实的方式代入，不仅增加了叙事的多义性，也因为多段文字欠缺主词，在修辞学上将观众纳入一种主观抒情的心境。影片中客观写实记录的文字运用与中国诗词及移情作用结合，有别于好莱坞依赖语言的现代功能，更多强调在视觉的抒情叙事上做文章，成为了忠实记录的工具，也是营造抒情美学的契机，让影片以缓慢的节奏弥散着散文般的抒情意境，形成了侯孝贤写实夹带写意这一独特的电影风格。

在结构上，经过后期剪接处理的景观更加强调了导演对影片节奏缓慢、简化情节和浓缩主观意识的把握。如前所述，影片还多次善用文字，出现完全无主词的对话，以此代替中国诗词，却又赋传统于其中。在镜头表达上有时更是将主观想象与客观叙述共同展现，比如林老师、何记者在宽荣的带领下一同看望文清，无法言语的文清用文字询问小川校长是否安康，之后宽荣回复写下的文字以旁白形式出现，得知小川校长因病逝世，走得十分安详，但静子小姐悲痛不已。仅仅一笔带过的静子小姐却成为下个画面的视觉主体，她与老先生一同静默地站在山间小路上。在此观众无法确切赋予这个镜头景观是写实或主观抒情，侯孝贤模糊了两者的界限并让其融合交织。《悲情城市》运用这类客观写实与主观抒情交融的镜头相当多，包括用四季轮回、生死交替的仪式这些平时时常发生的真实场景抒发中国人在历经政权交替后依旧渴望存活下来的生命力量，代入吃饭睡觉这些客观普通细节抒发中国人不断压抑、郁结于心的不安和痛苦，最后更通过几次吃饭场景的建立凸显真实中国背景下父权的交迭关系。

侯孝贤电影中的写实主义不追求商业电影的观赏性与娱乐性，加之导演本身具有的抒情特质，达到艺术上的创新和文化自觉，侯执导的影片皆通过写实与抒情融合的景观记录社会底层小人物的命运，记叙了一整部台湾文化变迁史。

# 第二章人物的描摹

人物的变化不会发生于真空之中，他们是环境的产物，一切人物都有其种族背景、社会背景、信仰背景和教育背景，所有这些文化状况都会大幅度地影响人物的性格，决定他们的思考和说话的方式，决定他们的价值观、关心的东西和情感生活。①悉德·菲尔德在《电影剧本写作基础》中将环境比作空的杯子，杯子就是背景，它是环绕着人物的空间，可以由故事和人物的细节灌满。②侯孝贤在电影中对人物群体的描摹尽量不加装饰的展现环境和故事，以此塑造人物在现实生活中本来的样子。

人物的创造开始于导演已经了解的东西。但是总体研究不一定能产生出足够的信息，还需要做特殊研究才能填补人物的细节，在导演自身的观察和体验里可能没有这些。③所以侯孝贤在结合自身经验的同时尽可能研究那个人物的人生轨迹，并在两人之间产生共鸣。侯孝贤从影片《儿子的大玩偶》开始，其中忧郁的气氛明显与其早期商业片讨喜、欢欣的气息截然不同。从《风柜来的人》开始，除《尼罗河女儿》《南国再见，南国》《千禧曼波》三部影片，侯孝贤在1980-90年代的所有作品主体都以自传体式的新电影清楚地呈现④。另外侯孝贤特别钟爱以李天禄的人生轨迹描写及边缘人作为题材创作，他自知李天禄宛若一座文化的宝藏。为了一部探索台湾历史的电影，李献出了自己的故事，个人的回忆唤起了历史分期、年代称号、朝代及描述时间架构的问题，侯孝贤透过李天禄找到了他需要的文化遗产。而以边缘人物作为电影主角也是侯孝贤电影的常态，如《悲情城市》中特别关注的女性问题。克里斯蒂娃（Julia Kristeva）在其著名的文章《女人的时间》中认为"认定历史的时间是直线时间；是像一个计划，呈有机式的发展，具有起、承、转、合等特性。而女性的时间则呈循环和纪念状态，可和不朽、永恒、再生和重复等特质结合一起。"在《悲情城市》里，男性的声音显然是用来做线性历史的工具，宽美代表的女性时间和女性书写则是另一种历史时间的形态，来讲述非指导性的个人历史。

# 第一节个人自传

随着1960年代二位数字经济成长率的影响，台湾社会父权传统家族认同上的情感压力开始转为内省，这点引发侯孝贤以私人历史和记忆的近距离去检视、探索过去。侯孝贤的电影表达是独特的，反动为静、反言如默、反虚入实、反西归中、反武以文，形成“无导演的导演”、“无表演的表演”，关注“前因后果无情节”的特征。①从《风柜来的人》开始，除《尼罗河女儿》《南国再见，南国》《千禧曼波》三部影片，侯孝贤于1980-90年代的所有作品主体都很清楚地以自传体呈现。

# 一、关注小人物命运

在法国、德国、日本与中国等形式各异的新浪潮运动中，传记均为一种主要说故事的形式。这些新浪潮时期电影尽管在文化、产业、风格及历史上不尽相同，但都具备反叛文化及电影传承的弑父情结。在各样的新浪潮中，我们都看到对传统类型的槟弃，而且为革新过去电影创立的形式，大部分新浪潮导演选择以自传体式为表达，符合新电影中败家子女的身份。自传电影也可视为新浪潮用来与过去电影时代做区隔的一个类型方式，自传电影的主题通常锁定在问题重重的童年、蔑视一切的青春、政治或社会巨变时的回忆。

自传，作为文学或电影的文本，都不再被视为"忠实、无染地对可证过去的历史重建”，而是根据文化、心理源头的一种"想像艺术"②。作为一种想像艺术，而非只是对往事的追忆，自传的真实性便在于“在复杂的自我探索、自我创造过程中逐渐形成的内容”；这里便是带入诗学，用以分析自传的功能与形势变化之所在。不管在个人或集体生命中，虚构、创造、编造事物都是真实性的中心建构。

台湾新电影同样如此，始于自我探索与自我表白。要了解台湾新电影中的自传体，关键在于个人对社会及政治变迁的回忆，导演和编剧在编写个人或彼此的故事时，也需要互相参照他人的故事。例如《悲情城市》里的摄影师林文清是参考真实人物胡宗辉而写成的一个主要角色，胡宗辉的回忆建构了该片记述家族、城市与家国历史的部分关键；《好男好女》则是将真实人生中一对伴侣的经历编入一个混合虚构、记忆、表演与历史记录的密实结构。因个人对家庭命运牵系时代动乱的省思，侯孝贤始终以包容的态度，真诚地关注着"个人"在时代长河的命运，通过电影为某个特定真实时代的真实个人书写属于他们的历史，在大江大海的时代波动中，让小人物的命运更显不凡。

在侯孝贤拍摄的一系列电影作品中，称为"台湾三部曲"的《悲情城市》、《戏梦人生》和《好男好女》分别讲述了不同时代的故事，这三部电影的时代背景贯穿着台湾在20世纪的苦难命运，包括市井小民、民间艺人和知识分子，分别以不同视角来看待每一段大历史，各自反映不同时代背景下的台湾。虽然侯孝贤习惯以史诗的方式呈现影像，但自传体式的带入并未影响整个格局，在侯孝贤试图勾勒“国家传记"时，自传的形式反而更加清晰。在侯孝贤的作品中，自传诗学是非常重要的概念。从很多方面来说，侯孝贤的电影生涯标志了台湾电影史过去近四十年的改变，但这些改变并不总是接续发生。在电影导演的个人经历与区域电影产业发展史之间，自然会有极大的分歧。①

早在20世纪80年代初期，侯孝贤试图将影片融入商业类型与本土感性，他在新电影运动中决定性的转往自传体式，将自传、社会与政治快速转型、乡土文学三者联系在同一个脉络中，从此侯孝贤的作者导演地位确立。侯孝贤的自传体在1990 年代"台湾三部曲"中达到高峰。这三部电影代表了台湾岛屿的人们(如《悲情城市》的林文清、《戏梦人生》的李天禄，以及《好男好女》的主角钟皓东与蒋碧玉），侯孝贤试图将他的自传形式嵌入台湾背景和人物本身的传记素材上，而这些电影的完成也标志着侯孝贤成为台湾最重要的导演之一，巩固了他在国际间的声誉。

随后，侯孝贤的作品《海上花》（1998）、《千禧曼波》 (2000）中也见自传手法的持续使用，这几部由侯孝贤御用编剧朱天文主笔的电影一边揭开个人回忆与经验主义历史的纠结，另一边与时间玩着奇特、感官上的游戏，即便到了 21世纪的《咖啡时光》 (2004），侯孝贤也巧妙地将1930年代著名作曲家江文也的故事融入一个现代日本女人的漂泊人生中。自传体式电影更新了以侯孝贤为代表的一众新导演与台湾电影复杂又纠结的关系，更为重要的是，自传体例为侯孝贤提供了如国家认同、女男两性本质、犯罪与乡里等更多台湾亟需面对的核心问题。$\textcircled{1}$

从1983年至1986年，侯孝贤以一年一部的频率连续执导了四部“成长电影”《风柜来的人》（1983），《冬冬的假期》（1984），《童年往事》（1985），《恋恋风尘》（1986）。按照影片剧情中年龄发展的顺序应该是《冬冬的假期》，《童年往事》，《风柜来的人》，《恋恋风尘》。这四部所谓的“成长电影"都来自于侯孝贤一众好友朱天文、吴念真，包括侯孝贤自己的一代集体童年、青少年记忆，毫无疑问皆是自传体作品。在处女作《风柜来的人》中，“坏孩子"的形象是侯孝贤带有自传性的表现角色，在角色的身上折射出侯孝贤自身的成长经历，他是台湾社会现代化以后被竞争机制抛弃的边缘人物，精神和实体上都是无根者，是都市的飘泊者，这是过去的台湾电影从未出现的角色类型。侯孝贤将这种人物类型引入台湾影坛的中心地位，用纪实性镜头和意识自传体角色将客观写实的事件融合在一起。与乡土角色与之对立的是都市角色，躁动不安着，充满罪恶与诱惑，青少年无可奈何地拔掉乡村社会的根，在都市飘荡，这是社会进步，也带来了罪恶，侯孝贤用不动声色的冷静镜头语言吐露了台湾乡村社会的解体，和都市社会精神飘泊者的无奈无助和郁闷。

在《风柜来的人》之后，侯孝贤意识到将自传性的角色代入电影所塑造的台湾社会正是他想要表达的，于是他根据合作伙伴朱天文和吴念真的生平故事拍出《冬冬的假期》（1984）和《恋恋风尘》（1986），此后又受到作家沈从文的启发，拍摄了自传色彩电影《童年往事》（1985），在90年代厚积薄发，积聚能量打造了《戏梦人生》（1993），由台湾国宝大师李天禄在本片中饰演自己。《电影手册》的主编Jean-MichelFrodon评价侯孝贤身上有一种很纯洁的气质，以致他能不同于常人，用另一种独特的眼光看世界，更受触动的是，侯孝贤的叙述方式能透过他的电影看到世界，看到自己。侯孝贤的电影能够让观众“看到自己”，拥有一种亲切感，全得益于他能够借助一系列“借来的”人物自传故事带入影像，透过自传体裁，不断探索着台湾电影中的文化遗产。

# 二、见证大时代变迁

这里主要以影片《童年往事》和《戏梦人生》来分析侯孝贤电影中描摹人物

的自传体形式。

《童年往事》虽是侯孝贤个人的自传，谈述某个人的故事看似单薄没有层次感，实则却因导演独到的处理方式赋予电影丰富的意义，在不断酝酿中打造台湾80 年代经济、社会、文化、政治不断转变的证言。

电影开场用一段旁白建立镜头与导演之间的故事：这部电影是我童年的一些记忆，尤其是对父亲的一些印象。这句话说完后，侯孝贤便在三分钟内描述了因战后政局转变，随着国民党政府大规模的迁徙，他的家庭也从广东来到台湾，由此开启追述阿孝咕一家自大陆迁台的经过。值得一提的是，在所有侯孝贤的作品中，这个开头是唯一可以听到他作为角色的发声，这段直接的陈词明显与沉默的目击者相矛盾。尽管如此，这部电影整体而言还是由沉默但悲悯的旁观者作主导，主要跟侯孝贤的"父亲因病缠身，大多时候保持沉默且很少走动"有关，侯孝贤的自传因此由回忆迁徙、动荡和转移、过度衍生的疾病而展开。这样的开局似乎也预示着这个家庭的历史注定要经历数次死亡和最终无法避免的崩解。这段主观叙述也带领观众进入侯孝贤的世界，与他在台湾的成长经历产生共通感，赋予电影回忆的基调。

导演陈国富评论《童年往事》，他认为电影中描述的这些死亡与记忆因放置于一个时代变动的背景中，所记叙的不再是导演的个人乡愁。纵观民国四十七年台海空战、陈诚逝世消息公布于众、最后做好一切准备渴望回返大陆的祖母等一系列情节描写，代表的不是侯孝贤的童年往事，更是那个时代每个小人物的往事记忆，是一个民族的历史记录。影片末了，四兄弟为祖母送葬，四兄弟代表了新一代的更替，是每一代新人对另一代人逝去的见证。①

在影片中，侯孝贤叙述来到台湾乡下的客籍家族史，他们一家居住在有榻榻米与纸门的日式房子里，思念着回到大陆而亡故的奶奶和罹患疾病而归天的父母。主角阿孝轱以打架度日，然而当他成长为青年，自此才知道家族的悲哀历史以及亲情家国的可贵。这种家国眷恋并非存在于阿孝咕个人的成长记忆，他的家人特别是祖母、他的朋友同学老师，还有无数像阿孝咕一样的人们，他们都是那个时代和地区的代表人。片中所表现的台湾社会正是日本导演是枝裕和从小从父亲口中提起的场景，“如果没有通过电影认识侯孝贤的话，我可能都不会成为一个电影导演”。①是枝裕和坦言，自己从影片《童年往事》里看到了父亲曾经一直给他描述的台湾生活，才明白父亲为什么总是在想念台湾，因此也在精神上将侯孝贤视作父亲一样的存在。片中的人物与人物的行为向我们呈现了一个自1949年以来的台湾生活图景，由老一代在政治文化上的本土意识的根深蒂固，中一代的惋叹与绝望，到下一代的台湾意识成长，清清楚楚呈现整个台湾世代变迁史，全片以阿孝咕的成长作引，勾勒出对这些年代的惋惜与悲哀。而这部电影由自传式题材变为时代记录的经典，更多是与结构情节完整，细节丰富细腻有关。《童年往事》虽是以散文式碎片化加之组合，但每个片段都发生在属于那个时代故事里并浑然一体，如全家人正其乐融融吃甘蔗却突兀地听到广播那头播放着米格机被击落的消息、少年在打桌球时与老兵发生冲突、晚上睡觉不时听到坦克驶过的轰轰声等，所有看似不相关的细节都触及台湾当时的社会敏感问题。正是这种强烈的真实感与亲近感，让影片不再只是成为记录真实的工具，更是招引那个时代的那些人对历史的回溯。

如果说在侯孝贤的新电影中，国民党是最主要的反派，那么下个目标就是殖民台湾五十年的日本，如何将日本统治台湾的悲惨时代用毫不虚假的方式投射到银幕成为侯孝贤当时最重要的事情。幸运的是，侯最终找到了台湾最受尊敬的民族偶戏阿公—李天禄。《戏梦人生》赢得1994年坎城影展评审团大奖后，戏中的主角李天禄也从此扬名世界。在侯孝贤多部电影如《恋恋风尘》《尼罗河女儿》中都饰演阿公的他，已是新电影的知名人物，他在《恋恋风尘》中饰演一个坚强的家族长者，于日本与国民党统治期间存活了下来，而现实世界中的李老始终助力保护中国濒临消失的木偶艺术。

《戏梦人生》中的李天禄，身为底层平民，他的观点有别于官方历史所讲述的真实，重点放在进一步探讨他身为布袋戏师傅这样一个特殊的社会文化角色对个人记忆和历史时空的意义。侯孝贤对李天禄在影片中再现或陈述个人与家族历史或大时代的历史事件时映射出历史交错时刻戏剧的文化转折，呈现殖民力量介入后对家国文化所造成的意义位移，“这部关于李天禄的影片也将这位布袋戏大师摆放在各个传统美学之间的交错点一人类生命模型和自然事件之间的辩证，也是真实与舞台、真实与想像之间辩证的交会之处"①，对于李天禄的人物描摹不仅是美学的，也是历史的。

# 第二节戏偶形象

中国传统的偶戏有两种一傀儡戏和皮影戏，布袋戏属于傀儡戏的一种。大约从20世纪20年代开始，在福建已经盛行的以武侠戏为主的布袋戏开始流行于台湾民间，从此布袋戏在台湾经历了不断地变化与特色化发展。布袋戏在台湾发展出自己的道路后开始成为台湾地方特色文化的一种表征，放置于不同时间空间与时代背景中，赋予着不同的社会文化内涵。②

内容丰富多样的台湾布袋戏在当时成为台湾三大偶戏中最富生命力的一支，其中以戏偶大师李天禄为代表，他是从大清到日本统治再到回归祖国这段全历史的见证者。在侯孝贤的电影中，通过戏偶形象李天禄这一独特视角记叙台湾地方戏曲、讲述台湾日占时期长达50年的兴衰枯荣。

# 一、传统文化的呈现渴望

相较于《悲情城市》通过一个家庭映照历史，侯孝贤在《戏梦人生》中更渴望呈现台湾和台湾文化上的问题。

于是，在影片《戏梦人生》中，侯孝贤只透过一个人物—李天禄，这位从《恋恋风尘》(1986）开始跟侯孝贤合作，也是台湾知名的布袋戏大师来言述历史，如同李天禄操作戏偶的绝技，侯孝贤在《戏梦人生》活用李天禄的功力，将他独特的个性发挥出来，用特殊的自传表现手法展现当时台湾人的家国心声。以《戏梦人生》这部用后设手法同时混融历史与记忆、文化与艺术、集体与个人的影片为例，侯孝贤借用布袋戏大师李天禄的身世指涉台湾近代史即为混乱认同政治的事实。从台湾新电影运动开始，侯孝贤就以自身经历和他人传记记述台湾本土主义的变幻，到《戏梦人生》则更甚透过李天禄的生平及其布袋戏的展演形式构筑出台湾电影文化遗产的精神。

《戏梦人生》中的时间跨度正是日占时期，从1895到1945年。最后影片结束在日本军队投降时，李天禄回忆到：这个好消息到来后，自己变得有多么忙碌，让台湾从占领中解放，各地邻里都想要请偶戏来演出，以对神明呼应祈求表达感谢。尽管李天禄那时受尽疾苦，他还是必须日夜表演。这反映了偶戏于庆典、特殊场合上的功能；偶戏的重要性在于表演，而不在实用的讯息或说教，因此核心是操偶大师与戏剧演员。民俗偶戏扮演的是传承技艺的主要角色，因此在主题或故事中的意涵反而不那么重要。这和宣传式偶戏的舞台呈现截然不同，民俗偶戏的宣传手段反而凸显出台湾民间艺术表演的丰富性正是有机的与社群连接在一起。①

# 二、与布袋戏的紧密呼应

几世纪以来，台湾一直是落后的殖民地，但也是宝藏之岛，不只是因为侯孝贤和杨德昌努力以电影在现代西方领域中建立特色，也因为传统民俗艺术家李天禄。

在侯孝贤拍摄《红气球》时，其中有一段是女主角茱莉叶·比诺什在戏里替木偶配音，侯孝贤对此曾与布袋戏作对比：布袋戏跟东方戏剧一样，格律很严；人物出场要有进场诗，而且动作像平剧，要跟鼓声走..我从小看很多布袋戏，所以才拍李天禄。 ②

《戏梦人生》的主题和形式结构与布袋戏紧密呼应。这部电影一共100个镜头，其中李天禄的镜头有6个，一共45分钟，观众从中景特写镜头观看李天禄的行为，夹叙夹议，说史讲古。在主题上以李天禄的一生和布袋戏密不可分的关系为叙事主线，李天禄最擅长的就是布袋戏，最能代表他身份的就是戏偶，交错穿插其个人、家族的历史和日本殖民台湾的历史，反映李天禄“人生如戏、戏如人生"的生活态度。李天禄宛若一座文化的宝藏，为了一部探索台湾历史的电影，他献出了他的故事。③透过当时的历史重塑来安置新的台湾身份成为追寻根源的需求。

寻根一直是侯孝贤的主题，只是台湾的根盘根错节，难以记录。李天禄两边的祖父母、不同的家族姓氏和自己的不同面目叙述着他的根源，而他剧团里宛然如真的戏偶们亦接续担任了诉说的角色。传统偶戏是台湾特有的文化遗迹，是在中国民族地方特色的传承，既传统又现代，曾被日本和国民党作为政府宣传工具，但如今仍然保有弹性、坚毅和生命力，如同李天禄本人。

布袋戏这种戏剧形式及其传统并非曲高和寡的高级艺术，而是地方艺术，起源于福建南方。在李天禄的回忆录中，他花许多时间谈论布袋戏的不同流派、拜访大陆名家们的愉快经验、发掘布景布置及戏偶工艺的不同技巧。虽然戏偶的生命必须透过操控者才能呈现，但它们的生命通常比人还长。李天禄的父亲（同样身为戏偶师傅）在死前把戏偶们传给儿子，这是他仅有的宝藏，戏偶是传统而永垂不朽的艺术，它带着李天禄渴望回到祖国的希冀下继续存活。另一方面，尽管宣传式偶戏的美学意义如此贫乏，但放在李天禄人生故事中却很重要；宣传式偶戏团虽然形式粗糙，却让李天禄能回到本业上。戏偶是侯孝贤在这部电影透露的主题，他拿这个主题做引，时刻贯穿描摹李天禄这一人物，以此代表当时大部分台湾人民的心声，渴望追寻根源，渴望透过历史重塑来安置新的台湾身份。

# 第三节 女性角色

女性角色在中国电影中具有不可替代的作用，但随着时间推移，女性在电影中所处的位置在不断变化。女性主义电影理论家劳拉·穆尔维 (Laura Mulvey)认为在电影馆内观看电影是一种窥视行为，电影在明处放映，观众在暗中窥视。这种窥视包括电影中男性对女性的窥视，电影中女性是被看的对象，同时她们也是镜头内外的男性欲望所投射的对象。①

# 一、配角位置的女性

女性是为衬托男性而存在的，这种现象在中国电影中屡见不鲜，这在侯孝贤的电影中也有这样一个持续变化的过程。

朱天文曾表示，从80年代开始，台湾新导演都或多或少会将焦点关注在与女人有关的事件上，侯孝贤过去是完全不懂女人的，在影片中展现女性视点更不可能，所以在他早期的作品中总与坏男孩和黑道有关，并且有些大男人主义。直到侯孝贤读到《好男好女》和《海上花》两部读本时，他开始从另一个面向了解女人。

尤其是《海上花》，侯孝贤是透过朱天文拍女人的，他逐渐发现女人在其过去的作品中是经常缺席或作为边缘人物出现的。"其实从侯孝贤的影片中可以看到他的女性视点跟随年代变化在不断转移。在1998年的纪录片《侯孝贤画像》（又名《HHH》）中，他曾袒露自己对于原始的、雄性的狗咬狗世界的怀念：“反而黑社会是雄性表征，比较浪漫，比较直接。我在台湾也一直感觉到这种原始。因为我年轻时候那段经历，我总感觉，在自己成长的地方，你没有做到最顶，你没有变成城隍庙那个地方的头，没有做到流氓的头，你会一直感觉不足。因为我感觉男人嘛就要像个男人，现在男人越来越中性。其实以后女性一定是要比男性要强的，这个世界会转变的，绝对的。而我还是比较怀念那个雄性的世界。"②

此时的侯孝贤的世界是被雄性荷尔蒙占据的，这时期创作的影片多半将女性角色置于配角，为男性角色提供服务，甚至仅是为了影片的连续和起承转合而出现。例如"黑道"在侯孝贤的早期作品如《风柜来的人》《童年往事》中都有隐寓甚至成为电影主题，而女性在《风儿踢踏踩》《儿子的大玩偶》中多以边缘人物刻画出镜。

# 二、女性主体的再现

能够加入台湾新电影运动的创作，侯孝贤无疑是十分幸运的。这时台湾电影界出现了“新女性运动”，为女性导演和影视中描摹女性角色的创作提供了有利条件。若是仔细观察侯孝贤的影片会发现一个规律，他所有的作品中出现的女性，无论从性格、语言、行为、人物关系来看，都是他儿时最常接触的母亲、姐姐和祖母这三个女人的再现。

在电影《冬冬的假期》中，其所呈现的乡土在地空间和事件都大量存在着性别议题的脉络，影片中段以女性角色婷婷与疯女寒子间的互动代入性别课题。在原著小说中，朱天文以安安作为叙事观点，婷婷这一人物显得有些平板被动，但在电影中，侯孝贤却为婷婷增加了一个对抗男性的剧情：某日冬冬与男性好友结伴溪边游泳，但他却拒绝婷婷跟去，男孩们脱光衣服在河里玩耍时，婷婷一气之下偷偷将男孩们的衣服藏起来，让他们窘迫地遮掩身子跑回家。这个相比小说新增的剧情提供了女性存在反击力的可能性，这一反抗让婷婷在叙事上也存在一个主体位置，建构出更为强烈的女性象征意义。 ①

《童年往事》以外婆塑造的女性角色最值得细究。在从前的时代里，导演的(也是主角阿哈的）自身经验都是一点一点自内心释出，因此存在着不能再回家乡大陆的事实的人，而外婆正是扮演这样一个过渡性的关键角色，她是唯一回到家乡的人，是唯一在小镇上四处呼唤孙子名字的人，是唯一把主角的家庭和他的青少年世界联系起来的人。这一女性角色实际引导了整部影片的主流，讲述的不再只是一个从大陆退居到台湾的家庭故事，而是将重要的历史事件镶在家庭背景里，再通过个人经验的生、死和成长来产生叙事的变化，让我们意识到历史只能从个人的记忆中发生的事件、地点、生活的经验中去了解，外婆成为了这段岁月的叙事者。

再观《悲情城市》中的女性角色，叙事发展节点和人文主义观看都主要通过旁白展现。其中最关键的是宽美日记的旁白，我们从中得知宽荣和文清即将赶去处理台北暴动，提醒着宽荣回归人与人之间的关怀。在文清被捕时，宽美又有一段平和而坚定的旁白：天理有无我不管，只要他好好活着。宽美这一女性角色在社会律法之外，带出的不是外加的社会责任，而是她对文清的个人关怀，更是看似柔软女子实则内心坚强。对比电影中的男性旁白，以日本天皇的放送和陈仪广播为主，皆为无意义的宣告，更突出电影的声音以女性为主导并透过女性旁白推动叙事进行。此外还有一场有关盗印日本钞票的叙事，为协调冲突，阿捡姐说出“天破都可以补”，藉由女娲补天的神话比喻女性具有支撑社会的力量。前文在第二章第二节“自然景观与人文景观的交叠"提到过，九份山景作为自然景观多次重复出现，而其中最后一次出现九份山景是配以文清被捕之后宽美的旁白，她娓娓叙述九份山景和心境，衔接的画面却是痴傻的文良坐在赌桌前傻笑，然后是林家一家老弱孤寡围桌吃饭，却人事已非。此片以宽美的叙述始，以宽美的叙述终，呼应九份四季更迭，大自然循坏运转的节奏，也映衬出女人的韧性和生命。在家族男人相继死亡、失踪、痴呆后，靠着女人把家族支撑延续下去。女性主义者克里斯蒂娃在其发表文章《女人的时间》中是这样看待女性与时间的关系：历史的时间是直线无法循环的，却始终生生不息，具有起、承、转、合等特性。而相较历史的事件，女性的时间是非直线可循环的状态，总是与永恒、再生和可重复等关键词联系在一起。①女性时间和女性书写是另一种历史时间的形态，可循坏、可纪念、可在女性视角下凝聚一个家国大时代。

经过80年代新电影时期的"新女性运动”，90年代明显开始从女性角度谈及社会政治运动，在身处混沌时局的女性展现了一种前所未有的生命力。直至千禧年，侯孝贤在电影中的视点明显发生转移，女性开始代替男性成为主观视点。在《千禧曼波》《咖啡时光》《红气球》到2015年的《刺客聂隐娘》，女性成为父权统治社会下的关键性角色。

影片《千禧曼波》可谓是一个极大的转折点，由舒淇饰演的女主角曼波开始产生性别自觉而非被动受周遭环境或人物行为改变，进行主观地判断与选择，这便与前期的作品划清了界限。戴锦华曾在《昨日之岛》中说道，“一个化装为男人的、以男人的、以男性身份成为英雄的女人，则成为主流意识形态中，女性的最为重要的（如果不说是唯一的）镜像”②，直到2015年的《刺客聂隐娘》，这部摄制跨越四年零一个月，使用胶片超过50万英尺的电影作品，主角聂隐娘不但是以女性为主体，更是跨越性别强调以人为主体，产生人的自觉非女性自觉。侯孝贤曾在采访中提到此部电影中的演员周韵，“她个性非常独立而且强的，是平的，是齐的。人家说男有刚强女性烈，她就是这种是平的，平等的”。③此后，“男有刚强女性烈”成为侯孝贤对待电影中的男女角色描摹甚至价值观的首要原则。

# 第三章历史的书写

台湾新电影艺术家是战后青年一代成长起来的80年代青年，他们大多本地出生和成长，他们在艺术上的集体反思深受张爱玲、沈从文、钟理和台湾乡土文化的影响，电影营养来自布列松、费里尼、伯格曼、安东尼奥尼和德国新电影代表的学院派精英文化趣味①。对琼瑶为代表的商业通俗爱情剧深表不满。对于台湾与传统中国的关系，侯孝贤的作品中总是透露着国恋家恋，带有一种强烈主体意识与文化感，用悲天悯人的冷眼俯视人间的艺术高度，还原性镜头与心理视角相融合，将历史与人性进行非戏剧化的真实书写。

侯孝贤在一众新电影导演当中是最关心台湾历史的导演之一。在他的电影当中，台湾认同是最大的母题，藉由回顾台湾的过去了解台湾与大陆、日本和西方国家之间的复杂关系，并勾勒出台湾当代的社会现状。他的自传电影《童年往事》超脱了成长故事的主题，进一步检视台湾当代生活的种种源头。②侯孝贤的电影作品有着不可动摇的意义，总是代表了某一重要历史阶段的任务完成。如影片《悲情城市》，它超越时代的禁忌命题，直接表现最大的政治禁忌“二二八事件”，如今看来影片中隐晦的观点显得明朗，侯孝贤的客观历史态度反映了一段真实的台湾历史，而他在历史大叙事中真正关注的无疑是那些在族群利益纠葛中善良的人民与质朴的生活感情和梦想，挥洒出一份沉淀浑厚的历史掌握与关怀。从《悲情城市》开始，不单标示出侯孝在他的"成长电影”后拥有越来越大胆成熟的客观写实美学风格，更代表了台湾新电影在本土历史探索和社会性揭露的重大性突破，为新电影于保守动荡的80年代末寻找到最有利的支撑和发展，为台湾文化的成长积累了宝贵经验。

# 第一节 缘何聚焦历史

第二次世界大战后，台湾于1945年回归中国，但因国民党的不作为，加上社会、经济等问题频生，以致外省人与本省人之间不断产生冲突和误会，也致使台湾此后持续四十年的灰暗政治环境。而70年代开始的乡土文学运动和80年代突然崛起的台湾新电影开始有意识的寻求台湾身份，扭转现实环境。这一时期是台湾影史上比较不寻常的一个环节，侯孝贤此时的作品开始探索一种新的电影美学，从摄影和叙述方面强调“远一点，冷一点"①，这些作品也是侯孝贤自己的童年故事，它们讲述的不只是创作者自己的亲身经历，而且同时也唤起我们共同成长的童年回忆。

侯孝贤在整个新电影运动时期持续聚焦历史，张扬表现本土意识，原因有二：

# 一、大陆作家沈从文的影响

面对沈从文对侯孝贤整个创作过程的影响，侯孝贤曾经解释在面对如杨德昌和万仁等导演们带回的西方模式时，自己是如何解决叙事形式的问题。

在一次访谈中，侯孝贤谈到了自己风格的转变：

在拍《风柜来的人》之前，我对电影的想法是蛮简单的：叙事，就是把剧本里的故事说出来，后来我认识了一些从国外回来的电影工作者，他们对电影有很多理论，我整个被弄糊涂，我很困惑，剧本已经好了，我却不知该如何将它成型，我的编剧朱天文在听我说完后，给我看了一本书叫《沈从文自传》，我读了之后，发现沈从文的观点感觉是从上往下看的，就像自然法则，它没有欢喜，也没有悲伤，我觉得这跟自己的想法很靠近，不管他是在描述一场残暴的军队镇压，或是各种不同的死亡，生命对他是一条河，不停的流，不停的流，无喜无悲；而结果是思想上某种程度的广阔，或是一种非常动人的观点，因为这样，它也就带了一种观点的宽容，而我决定要采纳这种角度，问题是要怎样将之移植至电影中，我其实并没有真的找到解决方法，但是我发现了一种简单的技巧，就是不断地告诉摄影师‘保持距离，冷静一点'，这让一些真实的情况能自然的显露自己，镜头只要停在一定距离之外，静静的看着它们。②

与其承认自己的不足，追逐西方的影子，侯孝贤走了另一条路——回到1930年代的中国，并在现代文学中找到他的电影期待。不同于同时代许多有名的作家或知识分子，侯孝贤从来没出过国，或学习任何外语。①从没“喝过洋墨水”，这点与沈从文相似。

同侯孝贤一样，沈从文幼年被乡土文化包围，童年生活自由冒险，父母对其疼爱宽容，让他在充满生气与淳朴的环境中成长，与北京这样的大城市产生距离，所以他的文学笔触更贴近于对田园生活的描述。当侯孝贤看到《沈从文自传》时，他发现沈从文的文字其实具有一种抒情特质，但却能将自身置之事外，一直用十分冷静客观的态度和角度来看待事物，作为一部自传作品，十分特殊的以类似第三人视角观看所描述的所有人与事物，观看是他永远不会厌烦的一件事，②并且作品中描述的都是些与沈从文生平贴近的小事，如对故乡与父母的记叙、对儿时认识的一个邻居的描写、对自己某天读的一本书的见解，以无数的生活小事凝聚了作者的人生，具有一种自由流动的气息之美。这给予了侯孝贤以启发，让他找到了一个属于他拍电影的方式。因为侯的成长经历与沈从文不谋而合，两人都是从小生活在乡镇远离都市，与身边满是学习西方文学理论的朋友无法达成共鸣，更是在当兵归来后渴望为自己的行业寻找一个新的出口。侯孝贤从《沈从文自传》中借鉴到这种不加以颜色、静默得描述感知到的一切事物，并将视点放在中国乡土这个最熟悉不过的主题上，形成一种亲切感，让侯孝贤在台湾新电影运动时期创作出一种独特且新奇的电影创作形式和电影观看方式，完全有别于其他饱读西方电影理论的电影人创作。

沈从文对侯孝贤的影响主要表现在影像上，在张兆和主编的《沈从文全集》中，沈从文曾提出人与自然“和谐共存"的理念，更希望流于自然。而当时正处于商业电影为电影主流环境的侯孝贤，在拍摄了一些青春爱情电影后，从沈从文那里学到一种更为达观的人生态度，逐渐靠近于对自然和生命的深刻体验。在电影《风柜来的人》中，主角阿清经历了离家、感情受挫、父亲去世，可这一系列生活的变故在侯孝贤的镜头下却显得云淡风轻，是随着影像自行散发的，更让观众在不经意间自主发觉。

1930 年代的作家满是困惑，迫切想为中国语言、文学及艺术找到新方向，对像侯孝贤这样前期妥协于爱情通俗剧，却愿望用写实主义书写历史的新电影艺术家来说，以探索1930年代契合1980年代是对过去文人电影的怀念，是重新对中国乡土进行演绎，是找到中国文学与自己的相契之处。沈从文与侯孝贤在两个时代并行着，一同演绎着别具一格的中国文学与电影创作。

# 二、台湾影人集体倾向的推涌

侯孝贤在整个新电影运动时期持续聚焦历史，张扬表现本土意识，另一方面则是来自台湾一代影人集体创作的推涌。

早在新电影未兴起以前，台湾商业电影是以逃避主义为大宗，银幕上充满强调感伤眼泪的爱情幻想剧，以及拳头枕头的功夫/黑社会暴力电影。这些夸大的幻想作品，极少触及现实。政治的扭曲环境直接影响到影视行业，电影检查十分严格的同时还要考量商业利润，在双重夹击下的台湾完全无法将真实的过去与现在呈现于银幕上，在这样的历史大环境下，侯孝贤慢慢成长着。但是，即使在他早期的商业作品中（如《就是溜溜的她》及《风儿踢踏踩》），我们还是可以隐约看到他对现实素材的关注，这让他逐渐成为一个自觉的艺术创作者，槟弃爱情通俗剧的创作，转而关注台湾历史与现状并进行有意识记录。

当侯孝贤在香港看到了台湾看不到的大陆片，他感动之余深受启发，感动的原因并非是影片的剧情，而是剧中人物讲话的语言和那些让侯孝贤念念不忘的大陆景色。因为侯孝贤从小受到的教育是中国大陆的教育，正是那时让他意识到故乡是磨灭不忘的永恒记忆，他希望能将故乡那番景象呈现出来。①台湾解严前后，社会运动最蓬勃，在台湾新电影运动的洪流中，新生代电影人不迷失方向，以自省与感悟拍出这块土地人民的身影，以回顾与抒怀来寻求文化与身份认同，疏离人际关系。这群有志之士以台湾成长经验为感受，创作出台湾新电影，各自运用哲理与视野，打造电影蓝图。80年代初期，《光阴的故事》（1982）和《儿子的大玩偶》（1983）这两部集体创作的作品正式代表台湾新电影于复杂的台湾熔炉中破茧而生②。其中《儿子的大玩偶》由侯孝贤执导，但实际上两部片子他都全程参与，在此之后，侯孝贤仍然保持高度活动力，积蓄能量为新人制片③，并寻求奠立另类制片组织的基础。电影是他书写历史的工具，在中国电影史上，像这样一贯而锲而不舍面对历史书写家国的创作者几乎是空前绝后。

1982-1987年是台湾电影界的突破性时刻，国际影展开始关注台湾，这股影响力持续至新世纪。他们以极少预算、执着热情、真诚态度和质朴影像感动台湾人民，也引起了国际共鸣。虽然“新电影"为期仅五年（1982年—1986年），片量仅58部（包括一些模仿新电影风格的影片），但一批批年轻导演务实革新不断跟进，其中就包括侯孝贤的电影《儿子的大玩偶》《风柜来的人》《童年往事》《恋恋风尘》以及《冬冬的假期》。电影评论家针对此潮给予肯定与鼓励，认为跳脱传统桎梏，提升艺术性语言，创造新气象。1983年在《儿子的大玩偶》中侯孝贤用长镜头调度场面，动作、对话、事件，全在深浅有序的画面里游移，表达人物心理与故事脉络，以喜剧形式讽刺历史大环境失序，流露人文关怀。

同样，侯孝贤努力记述台湾现代史面貌,想藉由影像揭露台湾四十年的症结，其中以《悲情城市》成为台湾的代表作，标志着台湾新电影真正进入成熟阶段。《悲情城市》作为首部在威尼斯影展获颁金狮奖殊荣的华人影片，为90年代逐渐蓬勃发展的台湾电影研究奠定重要基石。台湾新电影运动看似是场革新运动，海外影响深远，但他们在台湾有着难以磨灭的厚实基础。这些根基激发出新电影的创新形式，进而引起国际间的广泛关注，基础之一正是此前风靡全台数十年的国语片传统。几位1960、70年代的电影导演先驱，如李行、白景瑞、胡金钰和李翰祥等，受尊为“台湾电影之父”，他们的作品运用了早期的叙事、风格、产业组织模式，拍摄资金多半来自当局巨额补助，或华人跨国企业的投资。虽然这一批导演在台湾电影产业、类型和意识形态上受到牵制，仍然出现许多丰富高品质的电影作品。此时的社会背景无疑左右了新电影未来的发展，在限制住刚起步的新锐导演的同时也带来更多可能性。新电影骨子里承袭的创意实则是多方协调之下的产物，将电影产业和作者的主观意识全面整合，在互相竞争间寻求平衡。

# 第二节日常琐事记录

侯孝贤在时间处理上常以多重倒叙套层结构来叙述历史事件，让日常琐事在时间的流转中不断揭露。不同时点的经验透过多层次的传递，不断由叙述者和观众现在的现实感而产生意义的位移。①从“台湾三部曲"过渡到现代都会男女系列，侯孝贤演绎了个人经验、日常生活与社会历史之间相互衍生出的微妙关系。

从“往事三部曲"①、“悲情三部曲"系列足以突显侯孝贤作品对"成长”、“往事”“记忆”、“悲情"或"人与命运与时代”这些母题的强烈关注。对于侯孝贤而言，电影、人生、历史是无法切割的，他认为“其实我拍片的过程，即是我学习及认识历史、生活、人的过程”电影创造是其用以思考并呈现人和生活的艺术形式，而他喜欢将其置于较长的时间幅度来观察、探索日常琐事，经过时间历史的沉淀来回顾、检视其中的变化，借客观事件抒主观人文主义关怀，贴近生活平常。

# 一、倒叙手法的惯用

侯孝贤常在时间上习惯用多重倒叙手法来叙述政权转移与历史，他的电影中通过这种叙述方式来展现日常生活琐事或映射某历史事件，除了串联过去与现在，拼凑事情经过，也反映了叙事角色当下的观点和心情，而当下的时间点会因叙事角色的转换而前后变化，重构历史事件的意义。

影片《悲情城市》中出现多次倒叙手法的运用。在日本女孩静子即将被遣返前，她到医院将自己的和服和哥哥战死后留下的剑与诗交给宽美，并希望她将剑与诗转交给宽荣，随即一首日本民谣《红蜻蜓》缓缓响起，这时画面转向静子教孩子们唱《红蜻蜓》，而宽荣就站在一旁深情地看着她。此时在这段场景中出现的四个人物之间的关系变得十分明朗，静子与宽美的友情、静子与宽荣的爱情、宽荣与静子哥哥的兄弟情，都得以抒发。其后，画面再次切换，静子身穿和服插着花，隔壁房间的宽荣欣赏着静子哥哥临摹诗词，这时这首诗以字幕的方式出现在画面中。前段场景中出现的和服与诗在此都得到回应，通过多重倒叙的方式诉说着一代人的美好回忆与心酸往事。侯孝贤借此营造多重主体记忆，取代了突出场景与人物关系的方式，将重点放在那个事件中那个人物的观点之上，不断重塑那段人们记忆中的历史。

莫林·杜伦（Maureen Turim）谈到电影的倒叙，认为其往往模糊主观和客观,内心世界和外在现实，“穿梭连接个人记忆档案和历史记录”。③在这多重倒叙中，侯孝贤于影片中对于历史事件的个人经验和情感传达经常选择通过转交礼物、遗物、血书、诗文、照片等历史文化物件以抒情明志，伤感于在"小我"“大我”、“家”

“国”之间的矛盾与挣扎。倒叙手法的惯用沿着历史事件将片段式的日常琐事汇集，道出建构后的历史真相。

# 二、没有冲突的日常琐事

侯孝贤认为，冲突没有过多可描写，冲突并非真正的戏剧性。①长期研究台湾电影的学者詹姆斯乌登十分赞赏侯孝贤，他认为侯孝贤的拍摄风格使他绝对不会直接描写冲突，而是将摄影机对准事件冲突的前面或后面。②

侯孝贤十分擅长将"苍凉"感置于日常琐事记录中，在取材自真人真事的《好男好女》中讲述了一群青年为台湾奋斗的故事，背景从日殖到光复后再到韩战爆发时的台湾处境，涵盖40年代的抗日战争和50年代白色恐怖。影片中主角钟浩东与蒋碧玉体现出恋爱与革命乃不可分割的生命体，一群留日大学生不愿在台湾被日本统治，到广东惠阳参加抗日，辅线以现代女演员的心情记事、现代与历史革命两线交错而行。在戏中戏、现实与过往中交互叙事，探讨台湾子民对国家奉献与自我追寻。以凝重灰色调画面对应现代鲜丽的画面，以民族革命情感对比男女情爱的放纵，主角配角皆是特定时代的普通群体之一人，日常生活是不连续、碎片式的，正如真实生活中用断续的细节构建情感，展现时代女性面对情感的执着与勇敢。影片中讲述的三段故事年代不同，人物各异，稳健镜头彰显史诗力量。

电影在记忆与时间的建构中并行，碎片式的日常在《悲情城市》中更具戏剧性，道出林文雄家族的命运和历史不可分的悲剧。影片以林文雄为首的流氓所经营的"小上海"酒家开张为背景，藉由曾在上海为日军任通译的老三文良和小舅阿嘉的牵线，而涉入上海黑帮的走私贩毒和江湖恩怨，这部分叙事主要以“现在式”的动作影像和剧中人物对白来推动。历史悲情的部分则以1945年日本投降至1949年国民党政府自大陆撤退来台这段时期的政治事件为背景，讲述老四文清和朋友宽荣，还有以林老师为首的知识青年遭到迫害的经过，呈现了除“现在式"的动作影像和对白外，还有"过去式"的倒叙套层结构。《悲情城市》及其现象的社会文化意义不在其是否还原二二八事件的历史真相，而是此片在本土意识高涨的后解严时代，借着二二八的历史符号，在虚实夹杂的记忆重建中汇集官方历史论点、庶民观点、个人记忆，以及时代变迁的文化细节和日常生活点滴，见证着家国的失落，更投射了台湾社会在80年代后解严时期寻找家国和台湾主体的焦虑。片中几段日常情节展现当下大环境的力量，一群知识分子在酒楼慷慨激昂的高歌《流亡三部曲》：“九一八，九一八，从那个悲惨的时候，我离开了我的家乡，抛弃那无尽的宝藏。流浪，流浪，整日押在关内流浪，哪年哪月才能够回到我那可爱的故乡？”；黄先生的弟弟在被枪决前写下"生离祖国，死里祖国，死生天命，无想无念”；静宽决定留在山中，写下“我人已属祖国美丽的未来”，这份坚定与决绝是台湾人民根深蒂固的家国意念的体现，是台湾与祖国割舍不断的骨肉亲情。

我们看到，《悲情城市》中宽美通过口述日记与个人记忆琐事和二二八事件进行一番对话，来见证渴望回归祖国，但摆脱不了当局暴力和家族衰微的命运；《戏梦人生》中李天禄记录个人回忆来戏说日本殖民历史，作为历经家族疾病死亡和政权更迭的文化存活证据；《好男好女》的梁静透过代表现代科技的传真机以及蒋碧玉事迹的排演这些稀疏平常的事件，牵引历史的鬼魂，和自己的欲望对话。而身为导演的侯孝贤，则经历全球时空转化，徘徊流连于上一代的记忆与下一代的瞬间当下，在全球化资本主义市场、艺术电影的创作与第三世界电影期待的交错矛盾下，实现不同的电影美学形式，或由历史琐事引发当下对话，或由未来观点回顾现在，企图透过时间的沉淀，让台湾社会焦躁浮动、流窜着旺盛生命力的现状，成为历史的光影定格。

“个人记忆、文化与历史之间，并非疆界分明。相反地，记忆物往往从一个领域转到另一个领域，其意涵也得到转换。因此，个人记忆有时可以归属于历史之中，而文化记忆的内容也可以与历史并存”。①侯孝贤的电影弱化冲突却更而强调了戏剧性，被称为"侯孝贤三部曲"的这三部电影用日常琐事记录构成现代台湾历史上的编年史，从昔年日据时代开始到1945年的光复日，再到国民党来台后的"白色恐怖”，侯孝贤始终透过时间的沉淀，以未来观点回顾现在记忆，成为历史情怀的光影定格。

# 第三节 重大事件还原

侯孝贤的电影习惯简化情节，强调历史的叙述，透过视觉与听觉还原重大事

件，从而与观众产生时代或情感的共鸣。对于重大事件的还原，主要体现在侯孝贤对视觉和听觉运用的掌握。以《尼罗河女儿》《悲情城市》《恋恋风尘》三部影片为例，分析侯孝贤是如何运用听觉与视觉对重大历史事件进行还原。

# 一、视觉隐喻

影片《尼罗河女儿》中发挥了含蓄又准确的批判功能。这时期侯孝贤的关注点仍停留在市井阶层的市民问题。

青少年是本片的主角，他们的孤儿心态与侯孝贤的都市人间孤儿是一脉相承的，同样虚无且无力。影片中以黄昏夜景为背景，成为全片最重要的视觉隐喻，结构性地贯穿整个情节。随即出现的灰暗阴霾的台北天空，暮色中变幻的霓虹灯，重复有序的在每个情节不时出现。这完全是夜色迟暮的都市景观，所有的富庶和快乐的追求，都将"一步一步走向无光的世界”。

侯孝贤有意识地希望引起观众注意提出警告，于是我们看到那么多不协调且尴尬的场景，如阿三将自认为能彰显身份、拉风的吉普车停在中下阶层的山道旁，晓方身着一身名牌时装出入简陋的家中，甚至于他们挥霍金钱的意识——偷、抢、卖身是赚来这些钱的方式，只为了那一身现代人的时髦装扮和短暂忘却一切的谎言。

从《尼罗河女儿》中，我们读出了侯孝贤的无力感，对现实中的物质诱惑，对只有眼前没有未来的虚无，对青少年没有分辨价值观的能力，①这就是当下台湾都市生活的青少年现状。

# 二、视听交错

如果说《尼罗河女儿》仅仅将视点聚焦于当时台湾的都市社会事件，到《悲情城市》则开始藉由听觉和视觉呈现1945年到1949年期间的真实历史事件。

影片刚开始便可听到从收音机传来日本天皇的声音，宣告日本战败，画面则出现林文雄烧香拜祖，画外音传来文雄妻子生产的哀嚎声，接着一段字幕冲破画面突兀的出现在眼前：一九四五年八月十五日，日本天皇宣布无条件投降，台湾脱离日本统治五十一年。林文雄在八斗子的女人，生下一子，取名林光明。这段字幕显示新时代的来临，也似乎可以为林家带来新的转机，让他们从日本帝国的阴影中走出来。

但仅在两年后就出现了二二八事件，这件事情带来的影响渗入每个台湾人民的日常生活中。此时画面先出现过年舞龙舞狮的热闹场景，然后镜头带到空镜大远景，接着衔接到宽美工作的医院，医院工作人员正在一起收听广播。收音机那方传来陈仪的政治讲话，但画面呈现的是文清与宽荣即将赴台北前来医院和宽美告别。随后宽美日记用旁白代入，契合对官方说辞的质疑：“今天收音机说台北发生事情，台湾人和外省人相杀。...台北戒严了，大家都很害怕...一个战争才结束，另一个战争怎么会接着来？"陈仪的发言代表官方粉饰太平，安抚人心，却不断由视觉和听觉呈现的矛盾来戳穿这样的假象。①侯孝贤反复以人性残酷的历史对话来构建近代台湾人心灵史，通过人性向历史的责疑来反思台湾近代历史。

在视觉和听觉相互交错映射重大历史事件的同时，视觉和听觉又各自担当了不同的重要作用。

首先影片在听觉上承担了至少两个主要叙事功能：一是通过不同场景或段落间听到的内容过渡到下一场戏；二是在影像尚未转换前，下一个镜头提前由音效不知觉地代入前一个镜头中。不管是人声、音乐或环境的噪音，都构建了观众新的视觉与听觉经验体系。《恋恋风尘》中听觉主要用以突出声音艺术的前卫性，李天禄作为阿远的阿公形象第一次出现在侯孝贤影片之中，作为台湾本土文化的代表性角色，影片中多次出现阿公用台语唠叨的画外音，带来了浓重的乡土家庭气息。《悲情城市》的听觉部分主要以宽美的日记、静子的直接述说和阿雪的信件为载体，三位女性担当了声音的叙事者，女性的叙述在声音上较男性更为柔和抒情，宽美的叙事部分虽以日记体形式出现，但主要以叙景来说传，并非直接用白话表露心迹，风格更倾向诗话的内心独白。比如电影前部分有一个宽美乘轿上山的场景，旁白（也是画外音）道：山上已经有秋天的凉意，沿路风景很好，想到日后能够每天看到这么美的景色，心里有一种幸福的感觉。影片最后在宽美写给阿雪的信中，再次以旁白形式述景表情：九份开始转凉了，芒花开了，满山白濛濛，像雪。宽美的叙述在听觉上全部以声画不同步的状态出现，匈牙利电影理论家巴拉兹认为，“非同步的声音能独立于形象而存在，并对场面起一种说明性的作用。非同步声音的效果是有声电影最有力的一种手法”①，侯孝贤在《悲情城市》中通过这种非同步声音——即旁白代入画面，将主观情绪投射到外在客观世界②，藉由客体隐藏导演主观情绪，将历史重大事件潜移默化置于其中，女性更在这里成为整个家族的主心骨，为父权社会背景着上了女性色彩，提供多条叙述观点，更成为家族之间的调和剂，成为延续的精神力量所在，弥补了影视力量中女性发言权的空缺。事实上，女性在侯孝贤的众多后期作品中越来越受到尊重，过去甚至是现在，我们仍然很少看到一部中国电影对女性在传统社会的压抑委屈致以这样的同情。

侯孝贤对声音的运用与技巧之复杂是中国电影中都十分少见的，仅在《悲情城市》一部影片中就融合了国语、台语、日语三国语言以及广东话、上海话两种地方性语言，用贴近真实性的语言尽量描摹一个客观的叙事背景，通过语言的歧义性见证全片复杂政治背景和无法沟通的历史文化差异。

# 三、影像反复

侯孝贤在对重大历史事件进行还原时，创作电影的视觉相较听觉更需对影像的历史意义深入探讨。影片《恋恋风尘》中有几个影像反复出现：以大远景记录半山腰处拾级而上的人，人被虚化处理的同时用以强调突出环境；雾蒙蒙的海边隐约出现的山影；夜色朦胧时常有一人在捣鼓着支张着巨大的电影布幕；祖父一辈为即将入伍的孩子送行，孤独地放着鞭炮，向看不见的山下挥别。大自然成了侯孝贤说故事的一部分，而在这大自然中又寄托着导演悠然的深情。

《恋恋风尘》中小城子弟到都市打工受到伤害有苦说不出，却在天地中得到纾解。当阿远蹲坐在海边聆听阿公感叹台风影响农作的话语，当摩托车被偷，当女友变心另嫁带来巨大创痛，梦中思念家乡的景象成为阿远为代表的这些少年心灵的寄托。沉缓的镜头节奏却呈现着紧密的电影语言，比如阿云到裁缝店开始自己的第一次打工，信件成为与阿远两人之间的通讯方式，经常互相诉说着对家乡的想念，却因为这样的通信往来促使阿云与送信邮差间的往来，直接迎来影片最大的转折一阿远的失恋，而送信邮差的伏笔在不经意的细节中成为影响剧情的关键，正显示了侯孝贤不会在电影中刻意制造冲突，使剧情戏剧化的重要特色。

影片《悲情城市》同样如此，主角文清是一个聋哑人，在这种先天不足的背景下，导演却赋予他专业照相师的身份，借助具有记录性的照片象征历史的分期。此外电影还通过片段式的戏剧结构混入部分史实，用剧中虚构人物代替历史现实人物，如唐山何记者影射当时《大公报》的记者何康，林老师的角色影射钟理的哥哥钟皓东，以及陈仪的广播和真实的黑社会伪钞事件，还有像"父亲无罪"的血书这样借助剧情表露千千万万台湾人民当时的处境，这些都是对记载真实历史的还原，也使得观众与影像间产生复杂的阅读关系：历史、回忆、戏剧性、真实性混为一谈，虚实之间呈现出一种新的观影经验，①在还原历史事件的同时也以内省的方式透露侯孝贤“生离祖国，死归祖国，死生天命，无想无念"的真切心境。

# 结语

从20世纪80年代电影对台湾乡土文明和人格成长经验的追忆，到80年代末直面台湾人民迷失的惨痛历史记忆，侯孝贤的电影无论是从景观的呈现、人物的描摹还是历史的书写，他都在影片中展现浓郁的乡土人文气息，借自传立意，提出身份认同课题，更是将对大陆挥之不去的思乡之情一览无遗。他用作品启示台湾人民要正视历史、珍惜历史，从这段台湾的百年"悲情"史中汲取教训，以最大的宽容消解历史造成的伤害。

侯孝贤的电影具有一个十分重要的特质，风格淳朴且悠长，具有看似平淡镜头背后水到渠成的力量，作为台湾新电影运动的领军人，并持续影响着90年代台湾“新新电影"导演的创作，他的电影作品《童年往事》《冬冬的假期》《风柜来的人》《恋恋风尘》《尼罗河女儿》是电影作者心目中的经典，这几部成长电影塑造了一系列都市飘泊者的角色类型，展现了台湾地景实像的美好定格，透过事件所形成的经验构建出一个含蓄内省的时间与空间，而影片中展现在日常生活上的历史叙事与指引，缓缓地演绎着逐渐形成的生命经验。他的侯式创作倾向和自诉式的表现方式影响着一代又一代处于青春期青年人和电影人，找到了最佳的电影语言表达方式。直到《悲情城市》，侯孝贤正式确立了一种不同于以往的电影文本，受沈从文的影响，开始在带有强烈主体意识与文化感的基础上强调以悲天悯人的冷眼俯视世界，那种历史与人性、还原性镜头与心理视角的融汇，那种悲壮雄放的编剧精神，那种客观写实与主观抒情下书写的家国人生，始于他始终用一种东方式达观的人生态度，用即使在逆境依旧热爱生命与自然，尊重时代尊重历史，不断构筑他作品中的叙事态度。

作为一位认真、深沉、向往力量与激情、从生命的最底层走出去的台湾乡土电影人，侯孝贤也是一位具有情怀和丰富的理性思考的生命体验者，形成了属于他自己的成熟社会历史价值观，传达着人道精神与人文关怀，使他的电影不再局限于本省、外省和二二八事件的框架，从个人视角转而由内向外观察世界，关照人生大时代，成为一位坚定不移的电影艺术者。有些东西需要在你生命里沉淀，然后会在某个时刻发酵成形，映衬你的生命体验。正如他自己所言：“生命中有许多吉光片羽，无从名之，难以归类，也不能构成什么重要意义，但它们就是在我心中萦绕不去。"①这些"无从名之"的"吉光片羽”，牵引着侯孝贤拍出一部部电影作品，以不同的视角建构出不同的叙事，串联成让人难以释怀的母题，将个人与时代环境紧密地连接起来。

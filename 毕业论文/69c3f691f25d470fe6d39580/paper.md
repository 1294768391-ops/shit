合起来构成的力量推动着戏剧情境的产生和发展，同时也推动剧情的走向。随着真人秀的兴起，戏剧情境也被人为地放置在了真人秀节目中。

戏剧情境在真人秀节目中的重要性不言而喻，它能向观众清晰地展现时空环境、情节故事以及人物关系，甚至能激发情节里的矛盾。因此，为真人秀等电视节目设置戏剧情境是重要的一环。戏剧情境是戏剧动作存在的基础，戏剧是动作的艺术，戏剧动作是表现人物的基本手段。戏剧情境是表现戏剧冲突的前提和条件，人物间戏剧冲突的出现需要依赖于一定的戏剧情境。在特定的情境中，人物间的矛盾会受人物性格、动作的影响而爆发。

# 二、戏剧情境设置的多元化

人为地设定情境，通过节目中一系列游戏规则的制定，加上节目中人物更加丰富的行动线的交织，就会形成激烈的冲突矛盾和悬念。由此,情节会变得不可思议，使观众的观看欲望增加，情感被人物的命运而牵动。

首先，设置富有情节性的剧情环境。戏剧的时空环境，即情节继续发展所需要的时间和空间。时空环境主要指事件发生的大环境、时代背景。在这样的时空环境下，人物的关系日渐明朗，人物的冲突也得以激化，剧情逐渐达到高潮。戏剧中的环境是常规的、常见的。某人在某地做某事，这里的地点是环境，这个行为所处的时代也是环境。而推理类真人秀中的环境，来源于它们却不同于它们。《密室大逃脱》中，六位嘉宾所处的节目环境虽是具有剧情性的，但更是用来解密逃脱的叙事空间。每期密室的建筑面积都达到了1000平方米，而且里面的厉间层层相连,数量都超过了10间，而室内的每一种道具都可能引发机关，都是隐藏的线索。节目组将每一个逃脱的场景都主题化，再加上精心设置的线索和机关，增加了游戏的趣味性和真实感，让观众和嘉宾一起沉浸在这个逃脱的环境中，同时观赏性也得到了加强。

其次，设置任务和线索,推动人物行动。事件在戏剧中所起的作用是最为直接的，影响整个故事的元素一定包含着某一个具体事件，即便是过去式，矛盾冲突也会在戏剧的发展行进中进发。在戏剧情境中要有事件的填充让情境丰满，让人物关系明了，让人物性格鲜明，甚至改变人物命运。规则的设置能够影响人物的行动。《密室大逃脱》中，每期主题的故事都是通过嘉宾的行动来层层揭秘的，不同房间隐藏的线索组合而成的事件就是本期密室的主题故事。这些隐藏的线索有的是故事主人公的纸条，有的是房间里陈列的物品，还有的是除了六位嘉宾之外的角色扮演者这些线索都会让六位嘉宾沉浸在整个故事中，也让他们对解密有了思路，更重要的是让观众逐渐看清剧情，不再是云里雾里的旁观者。

最后，设置嘉宾形象，构成人物关系网。人物关系是指在一定的环境下所构成的单个人物与人物之间的冲突或结盟，构成剧作中人与人之间的联系。人物关系作为戏剧情境中“最有活力的因素”，是戏剧情节的重心，也是戏剧性的主要源泉③。人与人之间的关系多而复杂，有亲朋关系、矛盾关系等。在戏剧情境设置中，人物关系是否清晰明了决定着观众能否认同。只有塑造好人物形象，使其性格鲜明并颇具特色，才能活灵活现，抓住观众的心。在推理类真人秀节目中最离不开的就是“人”，即嘉宾。嘉宾的形象及嘉宾间构成的人物关系是节目的主要组成部分，人物关系的变化则推动着戏剧的发展。节目最终的目的也是展现嘉宾的个性、观念，以及在一定的人际关系网中嘉宾之间情感上的互动。

# 三、戏剧情境设置促进人物冲突产生

冲突和戏剧是密不可分的。没有冲突，就不会有戏剧产生。戏剧冲突是一种艺术美学的表现形式,在完整的戏剧行为中，体现为矛盾的产生、发展和解决。冲突是真人秀节目的内核力量，将人物放置在节目设置的大环境中，让他们去面对突发的事件，从而使节目变得好看和有趣。对于真人秀节目来说，戏剧情境中环境的设置会促进人物冲突的演变。人物的冲突大致分为三类，即过程冲突、人际冲突和自我冲突。下文将阐述三种冲突是如何被环境激发出来的。

一是节目流程激发过程冲突。真人秀节目中的过程设置会导致冲突，即过程冲突。真人秀的过程通常由节目组精心安排，具体内容提前设定。整个真人秀框架将使这一过程中的冲突变得更激烈、更紧密、更出人意料。从《密室大逃脱》一整季节目来看，过程冲突主要表现在解密的开头，也就是六位嘉宾戴上眼罩到摘下眼罩之间，这个流程是必须进行的。戴着眼罩面临的一切都是未知，嘉宾们不知道自己会去哪里，也不知道是六个人一起去还是分成几队，更不知道被带到的厉间里除了自己还有谁。节目流程的设置将嘉宾局限起来，为了故事的进展，他们不得不做出选择，从而激发了冲突。

二是生存环境激发人际冲突。真人秀节目包含了戏剧特性，若是没有人际矛盾的展现，整个节目的故事性就不清晰，好看度就会大打折扣。在《密室大逃脱》中，人际冲突被激发往往是遇到了生存问题，为了活下去而与队友发生冲突，或是与角色扮演者(NPC发生冲突。根据不同的逃脱内容，导演组设置的角色扮演者时而充当节目中的好人，时而充当坏人。当有了第三者介入时，嘉宾和他们就会产生奇妙的反应。有时是嘉宾出手相助，有时是角色扮演者的疯狂恐吓，有时甚至发生肢体上的对抗。除了嘉宾与角色扮演者的冲突，嘉宾之间的冲突也尤为明显。当六位嘉宾逃跑时，末位会被落下很多，甚至被堵在安全门之外。生存环境的改变自然会逼迫人物进行选择，而出于本能，人物会先把自己放置到安全区内。这样一来，人际冲突就会顺理成章地产生，并没有让人感觉生硬，反而会让观众产生共鸣。

三是游戏环境激发自我冲突。冲突是人物在某种特定的情境中所遭受的两种力量的分裂和对立。黑格尔认为冲突情境有三个层次，一是物理或自然情况所产生的冲突，二是由自然条件产生的心灵冲突，三是心灵性的差异而产生的分裂。前两种冲突是第三种冲突的基础和背景。嘉宾内心的冲突，很大一部分是通过规则制造来推动的。人与内心世界的冲突看似平静，但其实最具有戏剧性。导演组通过制定游戏单元的规则，让嘉宾做出选择，这不但能让节目更加戏剧化，而且能表现嘉宾在当时的游戏环境下的百变心情。在《密室大逃脱》中出现过一个最终抉择，六位嘉宾要从高台跳下至气垫才能获得最终胜利。在游戏环境的催促下，嘉宾源于内心的自我冲突往往是最具戏剧性的，是电视观众不曾预测到的。

# 四、戏剧情境设置的经验

《密室大逃脱》将戏剧情境运用得恰到好处，将戏剧情境的创造力和生命力发挥到极致。这使得节目拥有了活力，增加了节目的可看性和娱乐性,足以吸引更多观众跟随六位嘉宾一起解密。

# （一)应设置假定性情境

在真人秀节目中，为了对剧情中的现实进行加工，会设置戏剧情境，而这个戏剧情境同时也是节目组人为进行选择和设计的。在戏剧情境中，无论是突出节目中嘉宾与客观因素冲突的外部环境，还是努力塑造不同的人，形成不同人物之间矛盾冲突的情节故事，都是在节目组创建的虚拟环境中，在特定的情境中叙述和表达角色本身。六位嘉宾的逃脱环境破旧且不真实，机关设置简单，如果节目组没有设置假定性的情境，没有设置具体的节目规则，很难呈现戏剧性的情景，会让节目变得枯燥乏味。而节目组大量运用了情境中的假定性元素，对逃脱的故事进行了全面的解释,让简单的规则生发出人物之间的矛盾和令人意想不到的结果，推动节目的叙事进程，弥补人造环境的不足。

# （二)应突破人物关系塑造的屏障

若想把整个节目的剧本效果拉满，就必须抓好对角色嘉宾的选择和与角色嘉宾间所产生的关联的设定，并利用角色间关联的变化促进剧本的发挥，让整个节目规则呈现出最理想的效果。《密室大逃脱》节目组根据各嘉宾的相貌、性格、职业、年龄等客观因素，设定了相应的角色。嘉宾的形象经过设置后变得更加立体，六位嘉宾也能在节目中找到自我，快速融入情境中。此举不但使人物形象更加深入人心，而且使角色关系进发出真实情感，在保证收视率的同时，也丰富了节目内容中的矛盾点和冲突点。这档真人秀节目对人物形象的成功塑造,突出了节目中每位嘉宾的个性,而他们的命运故事也给节目创造了一种感觉，即生活就像戏剧，戏剧就像生活。这种戏剧性的角色设置使明星真人秀的连续性更强，有利于观众持续观看。六位嘉宾能否逃脱等悬念一直牵动着观众的心，让观众沉浸其中，节目的收视率大大提高。

# (三)应推动戏剧动作自然发生

戏剧情境中具体事件若设置得当，就不会让观众觉得节目的剧本感太重。作为推理类真人秀，推理大于娱乐才是正确的，仅披着推理外衣的真人秀则会失去情境设置中最重要的情境部分。许多观众在观看节目时不仅要求娱乐效果，而且想看推理时的填密，刻意营造的戏剧情境未必会受到观众认可。所以，在力求娱乐效应的同时也应把握好度，剧本化过于明显只会让节目的观感过于僵硬，让观众没有代入感。

# （四)应建构符合潮流的主题文化

为了吸引观众，追求收视率，很多真人秀节目过度刻画人物之间的矛盾关系，将冲突作为节目内容制作的重点和焦点，不仅没有达到预期效果，反而引起了网民的不满和批判，降低了节目的口碑。在满足广大观众娱乐需求的同时，真人秀节目要担负起社会教育责任，增加人文内容，使人文知识和戏剧情境充分融合，提高节目的含金量。在《密室大逃脱》的第一期《烧烤店的秘密》中，文化内涵被很好地融入戏剧情境中。节目讲述的是一个表面上开宠物店收养流浪猫狗，实际上是把这些小动物当成牛肉羊肉贩卖的串串店老板老陶的故事。节目的官方微博根据这次主题发布了一起"宠物领养”的公益活动，让更多人把眼光放到流浪猫狗身上，参与到公益活动中。这一行为与节目相联系，不突兀也不刻意，很好地传递了正能量。

# 五、结语

对于真人秀节目来说，合理运用戏剧情境是必要的，也是需要我们不断探索的。我们要看到戏剧情境中环境对推动人物冲突所产生的作用，也要看到节目中具体事件对人物动作的激发所产生的作用，更要看到人与人之间关系的变化和情感的突破。真人秀节目设置戏剧情境时，要保持正确的社会舆论导向，在情境中加入更多人文元素，并渗透当代艺术精髓，让真人秀节目向着更积极的方向健康成长。在电视受众受到多种媒介终端吸引而口味挑剔的时代，掌握类型节目创作的核心成为节目组的必备技能。通过优化戏剧情境，推理类真人秀节目将会以强烈的戏剧性和变化多样的形式感，展现出更加强大的生命力。

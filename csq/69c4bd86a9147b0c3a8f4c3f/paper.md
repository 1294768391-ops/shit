# 《哪吒之魔童闹海》中的传统文化传播与神话叙事逻辑

王亚杰

摘要：在全球化与文化多元的时代背景下，动画作为独特的艺术表达形式，肩负起文化传承与传播的重任。国产动画依托深厚的文化底蕴不断探索创新，其中以神话为蓝本创作的动画作品大放异彩，《哪吒之魔童闹海》便是典型代表。它借助3D建模、特效渲染等技术将古老神话故事以新颖的叙事逻辑呈现，并运用文化图腾、民族音乐等视听元素实现传统文化的现代化传播，推动着国产动画重拾传统文化，讲述着新时代语境下的“新神话”故事。

关键词：《哪吒之魔童闹海》；视听元素；传统文化传播；神话叙事DOI:10.19395/j.cnki.1674-246x.2025.09.002

在全球化浪潮的冲击下，文化多元性与独特性愈发凸显。动画作为一种跨越年龄与文化界限的艺术形式，不仅是娱乐的载体，更是文化传承与传播的重要媒介。尤其是国产动画，近年来凭借丰富的文化底蕴和独特的艺术风格，在世界动画舞台上逐渐崭露头角。其中以神话故事为蓝本创作的动画作品，如《哪吒之魔童闹海》通过巧妙运用视听元素，将古老的神话叙事与传统文化内涵进行现代化演绎，引发了广泛关注与热烈讨论。

中国神话作为中华优秀传统文化的瑰宝，承载着先辈们对世界的认知、对道德的理解和对美好生活的向往，蕴含着丰富的哲学思想、价值观与民族精神，是中华民族文化基因的重要组成部分。神话叙事逻辑是神话故事在结构、情节发展以及人物塑造等方面遵循的内在规律，它不仅决定了故事的连贯性和吸引力，更反映了特定文化背景下人们的思维方式和价值取向。

传统文化的传播对于一个民族的发展至关重要。在新时代，如何让传统文化在现代社会中焕发生机，实现有效传播，成为亟待解决的问题。国产动画以其独特的视听语言，为传统文化传播开辟了新路径。其视听元素包括画面构图、色彩运用、镜头语言、音乐音效等相互配合，营造出强烈的视觉冲击和听觉沉浸感，使观众更直观、深入地感受传统文化的魅力。

《哪吒之魔童闹海》作为国产动画的杰出代表，在神话叙事与传统文化传播方面取得了显著成就。电影以中国传统神话中的哪吒形象为核心，对传统故事进行大胆改编与创新，在叙事逻辑上打破了传统神话故事的线性叙事模式，采用更具现代感和冲突性的情节设置，塑造出个性鲜明、充满人性光辉的角色形象，让古老的神话故事在现代语境下焕发新生。

目前对于国产动画在神话叙事逻辑与传统文化传播特征方面的研究仍有待深入。以《哪吒之魔童闹海》为例展开研究，有助于深入剖析国产动画如何运用视听元素实现神话故事的现代演绎与传统文化的创新传播，为国产动画的未来发展提供理论支持与实践借鉴，推动中国传统文化在全球范围内的传播与交流。

# 一、概念阐述

传统文化是文明演化过程中逐渐汇集而成的、反映民族特质和精神风貌的文化，是民族历史上各种思想文化和观念形态的总体表征。在传统文化的基础上，传统文化传播被定义为通过各种渠道和方式，将一个国家或民族在历史悠久和世代传承中形成的文化元素、文化价值等向社会大众广泛传递，以此来实现文化传承、增强文化自信与文化认同等目的的活动。传统文化传播不仅是保存文化记忆、延续文化生命力的必要手段，也是增强民族认同感与文化凝聚力的重要方式。如全球华人共同庆祝春节、宣传中华民族传统服饰与习俗等，不仅增强了民族归属感，也强化了彼此之间的民族情结。

国产动画是由中国的动画制作团队或公司创作、制作的动画作品，尤其是动画影视作品，通过一系列技术手段将静态画面转为动态影像，包含二维动画、三维动画等多种形式。近年来，国产动画结合传统文化传播成为潮流，不少传统文化形象被搬上银幕并进行动画呈现，如以孙悟空、哪吒、姜子牙、白蛇、李白等为主角制作的《哪吒之魔童降世》《哪吒之魔童闹海》《白蛇：缘起》《长安三万里》等动画电影广受大众欢迎与喜爱。在这一“国潮"推动下，国产动画创作与制作技术日益精进，也催生了传统文化传播与民族文化精神的新颖化呈现。国产动画的传统文化传播意义重大，孙悟空、哪吒等传统文化形象在动画电影中得以被保留、被记忆，避免在时代变迁中被遗忘，增强了传统文化符号的生命力。3D建模、特效制作等现代动画技术为哪吒、白蛇等赋予新的视听呈现，观众看到熟悉的传统文化形象在动画中焕发新活力，会产生强烈的文化认同感和归属感，不断增强民族自豪感与文化自信。同时，高票房和丰富多样的周边产品也说明以传统文化形象创作的国产动画作品具有极强的商业价值，不仅推动了动画产业的繁荣进步，也带动了各类相关产业的发展，为社会经济注入活力。

国产动画的神话叙事是国产动画以中国传统神话故事为素材或灵感来源，运用动画的表现形式来讲述故事内容、传达思想和情感的创作方式。①在叙事内容上，它主要取材于中国丰富的传统神话宝库，如盘古开天、女娲造人，以及佛教观音故事、道教八仙传说等宗教神话，还包括牛郎织女等民间神话故事。这些神话和民间传说为动画提供了大量的人物形象、故事情节和世界观设定，如动画《宝莲灯》就取材于沉香劈山救母的民间神话。在叙事特点上，国产动画的神话叙事往往包含着浓厚的文化底蕴和丰富的传统哲学思想。在情节构建上，除了利用神话原本的想象空间外，还会设置曲折离奇的情节，丰富原本的神话传说内容，如《白蛇：缘起》以白蛇传为蓝本，加入小白与许仙的前世故事。在人物塑造上，除了借助神话所赋予的超自然能力外，还会赋予人物独特、鲜明的人物性格，如《姜子牙》在形象塑造中融合了道教传统观念与理想人格，将姜子牙塑造为坚守正义、追求真理的智者与仁者，使人物更加丰满突出。

在国产动画的不断创新与制作中，以孙悟空、哪吒、姜子牙等人物为主角的国漫神话叙事被逐渐构建起来，将古老神话以动画形式呈现，使得传统文化在新的时代语境中传承发展。同时，基于现实议题对神话进行改编与再创作，为传统神话注入了新活力，如《哪吒之魔童降世》对哪吒传统形象的重新诠释，“我命由我不由天”的主题契合了当代年轻人的精神追求，使传统神话焕发出新的魅力。国产动画神话叙事的不断发展，体现出中华优秀传统文化传承呈现方式的不断成熟与完善，也彰显着我国文化自信与民族自豪的提升。

# 二、《哪吒之魔童闹海》中传统文化传播的视听呈现

# （一）传统文化传播在视觉上的丰富呈现

# 1.传统文化中视觉的符号运用

中华传统文化对于不同的色彩有着不同的寓意，这种意义赋予成为一种象征符号，影响着人们的环境感知和情感认知。《哪吒之魔童闹海》的整体色调以中国传统色彩为主，如红色在中国文化中象征着热情、勇气与生命力，影片中哪吒的红肚兜、混天绫等红色元素贯穿始终，强化了哪吒勇敢无畏的性格特征；陈塘关以橙黄色调为主，契合中华文化中温暖活力的文化寓意;海底龙宫以蓝、紫色调为主，与传统认知中深海的神秘、幽冷印象相符合；在战斗场面中，电影大量运用对比色和互补色，如哪吒的"红"与敖丙的"蓝"，借鉴传统艺术中的色彩搭配手法增强视觉冲击力。

数千年来的神话与传说为中华民族留下了悠久的精神文化，这些精神文化往往运用传统图腾符号进行表达，这在《哪吒之魔童闹海》的制作中也被大量运用。如敖丙的"万龙甲"由全体龙族的龙鳞制成，体现出龙族的团结与肃穆;李靖与殷夫人盔甲上的饕餮纹是商周时期青铜器上的常见纹样，图案庄重神秘，契合了影片的神话背景；哪吒服装上的莲花纹样在传统文化中具有高尚、纯洁、神圣的象征意义，与哪吒的神性、纯洁本性和超凡脱俗的气质相符合。

中华传统文化中的图腾、文化符号与色彩寓意是中华文化的精髓，这些元素承载着数千年来人们的集体记忆与情感。《哪吒之魔童闹海》以生动有趣的形式传承人们熟悉的图腾、符号、色彩等文化元素，会让受众产生强烈的情感共鸣，仿佛置身于传统文化氛围中，更易与电影角色及情节产生情感连接。同时，观众在欣赏电影的过程中也会潜移默化地受到传统文化的熏陶，增强文化自信与民族自豪感。

# 2.画面中传统美术风格的运用

动画的整体画面对传统美术风格的运用，也是《哪吒之魔童闹海》体现民族特色与神话意味的重要方式。③电影的整体画风极具中国传统水墨韵味,如山水、云雾、亭台楼阁等场地充满古典园林特色，在特效的制作上也融入水墨画中虚实结合的技法，让画面在传递传统文化意味的同时充满层次感。在哪吒幼年回忆的片段中，镜头采用类似剪纸动画的扁平画风，在致敬传统民间艺术的同时体现回忆片段的故事感。在字体上，影片及海报中字体、字符等使用书法，或具有书法风格字体进行书写，也与影片整体的古朴、典雅的神话风格相契合，增强了整部影片的文化底蕴与视觉效果。这些传统美术风格承载着中华文化的独特审美观点、哲学思想与艺术技巧，使观众产生强烈的文化认同感与归属感，让观众在欣赏电影的同时感受本土文化的独特魅力，激发对中华传统文化的热爱。

# 3.人物形象塑造中的传统元素

在人物塑造的过程中，《哪吒之魔童闹海》也使用了诸多传统文化元素，这些传统元素在人物性格体现、故事情节塑造上发挥着重要作用。在妆容形象上，哪吒的“烟熏妆"借鉴了传统戏曲中“丑角""武生"的脸谱元素，眼部夸张的线条暗含"魔童"亦正亦邪的气质；在服装纹饰上，敖丙的服饰融合龙鳞图案和传统刺绣工艺，凸显了敖丙"龙太子"的高贵身份;在角色武器上，哪吒的混天绫以红色为主、以仙绦为形，火尖枪带有火焰图案，体现着哪吒的力量与灵动。这些丰富的传统元素不仅彰显着中华传统文化的丰富悠久，更能够帮助观众感受人物性格，增强故事的感染力，在推动故事情节发展的基础上更加深刻地表达电影主题。

# （二）传统文化传播在听觉上的丰富呈现

# 1.采用传统民族音乐渲染情节氛围

对传统民族音乐的运用是《哪吒之魔童闹海》的一大亮点，通过丰富的传统民族乐器演奏使电影的东方神话特色独树一帜。在乐器的使用上，《哪吒之魔童闹海》运用埙的沧桑音色渲染旷古神话世界的苍茫与威严，运用唢呐吹奏与锣鼓、竹笛结合展现战争的激昂。在特色演唱方法上，运用侗族大歌的复调唱法来体现"七彩宝莲"的圣洁与神韵，运用蒙古族呼麦展现"天元鼎"的威慑力、穿透力与邪恶感，用天津快板的风格特色来体现陈塘关的市井气息与喜剧效果。同时，电影动画音乐在创作中将传统音乐元素与现代电子音乐相融合，在融合中彰显传统音乐的魅力，进一步增强国产动画的传统文化氛围。

# 2.运用方言强化情节效果

传统的影视作品往往大多选择标准普通话作为主要的语言风格，但《哪吒之魔童闹海》使用方言，既符合角色自身的人物设定，又为剧情增加了诸多喜剧色彩。如太乙真人使用四川方言配音，既符合神话传说中太乙真人在四川江油乾元山金光洞修道的传统设定，又塑造出太乙真人与众不同的个性特点。在四川话的影响下，太乙真人区别于原本仙风道骨的传统形象，呈现出诙谐、慵懒、散漫的一面。“你打我撒"等四川话为影片贡献了诸多笑点，让太乙真人的形象更加鲜活。太乙真人的四川话，在幽默感中传递出四川文化的独特魅力，也使原本充满距离感的神仙角色更加"接地气",让观众更易产生情感共鸣。

# （三）视听元素相互交织下的传统文化呈现

视觉元素和听觉元素的完美配合是电影呈现出最佳效果的桥梁。在《哪吒之魔童闹海》中，传统音效与具体画面的高度适配为整个电影节奏的完美呈现提供了坚实的保障。如在紧张战斗场景中，激昂的鼓点配合快节奏,增强战斗的紧迫感;抒情场景中，悠扬的二胡旋律烘托出人物的细腻情感;诸多海浪声、法术释放声等模拟音效逼真，与画面动作完美结合，增强了影片的真实感和沉浸感。

镜头场景与音乐效果的结合，也充分实现了电影场景画面的烘托与情绪情节的刻画。如太乙真人施展山河社稷图法术的场景中，镜头中绚丽奇幻的景观与空灵的音效相配合，将观众的注意力吸引到视觉重点上，增强了场景的奇幻效果与感染力。青山绿水、云雾缭绕的视觉画面与古筝、竹笛的悠扬音乐构建出传统山水画般宁静、悠远的意境，还原了中华文化中的诗意。

将水墨画风、细致建模的视觉优势与传统乐器、特定音效的听觉效果相结合，不仅精准烘托出每一帧画面的氛围与效果，让观众感觉仿佛置身于神话之中，还为整部电影情节的走向做了充分的铺垫，推动电影剧情不断走向高潮。

# 三、《哪吒之魔童闹海》的神话逻辑叙事

（一）传统神话的解构与国漫神话体系的“新生”

在宏观视角下，《哪吒之魔童闹海》打破了固有的传统叙事框架，进一步丰富了在想象力下构建的"国漫宇宙"时空。③在情节设定上，《哪吒之魔童闹海》舍弃传统神话的家国叙事，大胆进行改编，采用以哪吒为主角的个人叙事视角，以"我命由我不由天"的主角精神为核心，讲述着发生在阴差阳错成为"魔丸"的哪吒身上的与命运抗争的精彩故事，丰富了传统神话故事的表达维度。

在创作构思上，《哪吒之魔童闹海》构建了一个庞大而完整的神话世界观，不仅有陈塘关、龙宫等经典画面的具体场景，还涉及天庭、元始仙尊、十二金仙等更为宏观的神话架构，为众多神话角色的登场、故事的开展乃至"国漫宇宙"的打造提供了广阔的叙事空间。

在创作拓展上，《哪吒之魔童闹海》加入了无量仙翁等新角色，进一步拓展故事情节，展现哪吒的新经历与新挑战，使哪吒的故事形成连贯且不断发展的情节体系。同时，电影还为其他神话角色如姜子牙等的出场埋下伏笔，并反复提及"封神大战",暗示了一个更大的神话体系的存在，为后续的故事拓展与角色联动提供了新思路，也为国产动漫神话体系的构建奠定了充分基础。

# （二）微观视角下角色故事的“新”审视

《哪吒之魔童闹海》打破了传统神话中边缘性角色本身故事单薄的缺陷，从哪吒角色切入，塑造了以哪吒为核心的系列人物的鲜明形象。在角色塑造上，将哪吒从传统故事中"灵珠子"转世的正义英雄,重塑为"魔丸"转世、顽劣的形象，与哪吒自身的纯粹特质形成反差和铺垫;将敖丙从传统的反面形象改为"灵珠"转世的孤独少年；将太乙真人从传统神话中仙风道骨、气质超凡的智者形象改为幽默风趣、不拘小节、憨态可掬的“胖老头”;将申公豹从传统神话中嫉妒心强、助纣为虐的负面形象改编为勤勉努力、备受打压、经历丰富、内心复杂的立体形象，引发观众对角色人物的共鸣。在角色心理上，将故事发展与角色性格相匹配，通过展现哪吒内心的矛盾挣扎与成长蜕变及其与父母、朋友的情感关系，挖掘出哪吒和其他角色丰富、复杂的内心世界，打破了传统神话中神仙不食人间烟火、不染七情六欲的距离感，让观众看到神话人物更具人性、更贴近现实的一面。

这一系列的颠覆式人物改编使得角色形象更加饱满立体、故事情节冲突跌宕，增添了“封神"这一故事背景的丰富性，为"国漫宇宙"下的神话人物建立合理且独具特色的个性化形象。④《哪吒之魔童闹海》解构了原本非黑即白的角色逻辑，让每个人物角色被理解、被看见、被共情，直面人性复杂与社会矛盾。

# （三）现实社会议题的映射与民族精神内核的契合

# 1.偏见与歧视下的"抗争精神"

在《哪吒之魔童闹海》中，哪吒因魔丸转世怕被发现而服用"迷迷眠眠丹"隐藏身份，申公豹、敖丙因妖族身份而备受偏见，“人心中的成见是一座大山"这一观点贯穿电影叙事的整个过程，也成为主角们抗争命运的内心映射。哪吒不畏天命、奋起抗争的精神，与中华民族自古以来面对苦难和逆境时不屈不挠、勇于抗争的精神内核相契合，从古代神话中的夸父逐日、精卫填海，到近代以来中华民族抵御外侮、争取民族独立的斗争，都体现了这种千百年来延续至今的抗争精神。

在现实社会生活中，人们常因为外貌、身份等因素对他人产生刻板印象或不公平对待的现象，而哪吒的那句“我命由我不由天"不仅是对自身命运不公的呐喊，也是对遭受不公群体的呼吁③，鼓励人们不被外在和环境限制，发扬中华民族千百年来不屈不挠的抗争精神，勇敢追求自己的人生，主动掌握自己的命运。

# 2.教育中的"爱"与牺牲

教育是一个亘古不变的"热门"话题，《哪吒之魔童闹海》中李靖夫妇、太乙真人等对哪吒的教育方式也展现了教育关系中的矛盾与温暖。李靖严厉、不善言辞的深沉父爱，殷夫人陪伴、理解的温情母爱，以及他们为哪吒所做出的牺牲，如李靖在陈塘关守护哪吒肉身、殷夫人刺骨疼痛拥抱哪吒等，都反映了现实中父母在教育孩子过程中的困惑、付出与期望。

李靖夫妇为了哪吒不惜牺牲自己的生命，哪吒也最终理解父母的爱并承担起责任，这体现了中华民族重视家庭、亲情传承的传统美德。在面对苦难时，家庭成员之间的相互理解、包容与牺牲，以及家庭作为坚强后盾的支持力，是下一代茁壮成长、不断延续的根基。这种浓厚又深沉的家庭观念是民族精神内核的重要组成部分，也是中华民族传承发展的情感共鸣。

# 3.放下恩怨的集体主义担当

《哪吒之魔童闹海》中有多个放下个人恩怨携手对抗的情节画面，如哪吒与敖丙放下命运恩怨联手战斗，申公豹、敖光放下与李靖、哪吒的恩怨携手对抗无量仙翁等情节，都体现出以大局为重的集体主义精神与担当意识。在中国的历史发展进程中，对个人利益服从集体利益的坚持铸就了中华民族生生不息、历久弥新的民族精神。在国家和民族面临危机时，无数英雄豪杰挺身而出，舍小家为大家。这种集体主义和担当精神在《哪吒之魔童闹海》的高潮情节中得到了生动、形象的体现。

《哪吒之魔童闹海》融合现实社会议题，引起观众共鸣，又以民族精神为内核唤起观众内心深处的集体荣誉感与民族自豪感，引导人们在面临社会矛盾与压力时做出积极、正确的抉择与努力，团结一心、勇于担当，共创民族复兴伟业。

# 四、视听元素下神话叙事与传统文化传播的相互作用

（一）传统文化呈现方式丰富化和具象化神话逻辑内核

建构在想象基础之上的神话逻辑往往抽象而深邃，其世界观架构难以被受众感知和理解，故事表达难度相对较大。《哪吒之魔童闹海》借助民乐、文化符号等丰富的传统文化呈现方式，帮助受众理解叙事情境、感受叙事氛围。在听觉的传统元素呈现上，运用唢呐、锣鼓、竹笛等传统乐器与交响乐混合搭配，如哪吒与敖丙合击时的配乐、七彩宝莲盛开时的音乐、激昂打斗场面的音乐等，这些恰如其分的音乐使用使得具体情节画面更有现场感和感染力。在视觉文化符号的呈现上，《哪吒之魔童闹海》将太极八卦、水墨丹青等传统元素与现代特效融合，如雷暴特效暗合道家“焉化万物"的宇宙观，陈塘关的街道布局所采用的中国古代城镇典型的方格网状结构彰显了古人对秩序和谐的追求。

人们对于历史、文化的感受往往源自具体且独具特色的视听符号，如悠然丝竹音让人想到盛世繁华，锣鼓声让人联想到激烈战争等。因此，这些传统文化元素为《哪吒之魔童闹海》的神话逻辑表达提供了极强的助力，让观众在具体的视听感受中融入电影情节，沉浸式感受主角视角下神话世界中的奇幻经历。

# （二）神话逻辑深化传统文化传播的力度与效果

以《哪吒之魔童闹海》为代表的神话逻辑叙事虽然相对抽象，但具有较强的内容咀嚼性，能够实现观众“常看常新"的效果，观众可以从反复的理解、回味中思考出更加力透纸背的民族精神内核与文化价值观念。这只有具有极强构建性的叙事逻辑参与才能实现，普通的歌舞、民乐难以达到如此的深度传播效果。

从《哪吒之魔童闹海》的具体神话叙事逻辑及情节体现来看，在儒家传统的视角上，强化了哪吒与父母的情感羁绊，李靖的隐忍父爱与殷夫人的"慈母"形象符合传统儒学的家庭传承思想；哪吒从反抗命运到守护苍生的责任转变，也暗合儒家“修身治国平天下"的传统价值观。在道家传统的视角上，《哪吒之魔童闹海》通过哪吒逆天改命的故事,将道家"盗天地之机"的修行智慧转化为大众可感知的成长叙事，以魔丸与灵珠之间的共生关系和不断变幻的动态冲突来展现道家"福祸相依"的辩证思维。

极具脑洞的神话逻辑叙事为传统文化的传播提供了新的途径，让传统文化以生动有趣的形式走进大众视野，尤其是吸引年轻一代。年轻观众在梳理神话逻辑的同时也会对其中丰富的传统文化元素有更多的理解与思考，电影会推动他们更加了解和喜爱传统文化，成为文化传承的主要力量。以《哪吒之魔童闹海》为代表的国产动画将传统文化以受众喜闻乐见的方式进行传播，有助于实现传统文化代代相传。

# （三）丰富的视听呈现为神话逻辑与传统文化传播“添翼”

流传千年的神话故事及传统文化的传承需要不断地创新，只有通过富有新意、喜闻乐见的视听呈现方式进行包装、塑造，才能被更多的观众听见、看见，而建立在3D建模、特效制作、多重音效等技术之上的国产动画就是新时代观众所喜爱的新方式。

《哪吒之魔童闹海》通过3D建模、渲染等技术构建出玉虚宫、陈塘关等宏大奇幻场景，以及太乙真人的各种法术特效，让观众直接地感受奇幻的神话世界，为神话逻辑的展开提供了令人信服的空间基础。利用动作捕捉和表情动画技术，赋予哪吒、敖丙等角色丰富生动的神态和动作以支撑神话角色的自身特性，如哪吒在魔丸状态下的火焰特效环绕，凸显其魔性和强大力量，契合神话中对其特殊身份的设定。同时，特效技术将神话中的神奇情节视觉化，多重音效使神话中的情节听觉化,如哪吒与敖丙大战时的各种法术碰撞、能量爆发，让观众清晰理解神话故事中的矛盾冲突与情节转折，从而实现神话逻辑在快节奏剧情中的顺利推进。

《哪吒之魔童闹海》将传统神话故事中的人物、场景、法器等元素通过现代视听技术逼真呈现，让观众尤其是年轻一代对于传统文化有了更直观的认识和亲近感。用现代的动画技术和叙事方式对传统神话进行改编和演绎，如哪吒的"不认命"精神与现代价值观融合，结合时尚、酷炫的视听效果吸引年轻人，使传统文化在现代社会中焕发出新的活力。同时，《哪吒之魔童闹海》借助视听技术的感染力，将传统文化中的正义、勇敢、家庭观念等价值观传递给观众，如哪吒与父母的情感冲突与和解，引发观众对家庭与亲情的共鸣，也推动着传统文化中的价值观在受众尤其是年轻一代心中的不断深入。

# 五、结语

《哪吒之魔童闹海》通过精妙的视听元素运用，构建了宏大且独特的神话逻辑，展现出生动鲜明的叙事特征。影片将传统文化元素融入视听语言，成功实现文化传播，同时借助视听元素强化神话叙事，塑造出丰满的人物形象和精彩的故事情节。视听元素与神话叙事、传统文化传播之间相互影响、相互促进，为国产动画电影在文化传承与创新叙事方面提供了宝贵经验与广阔空间。未来，动画电影创作应进一步挖掘视听元素的潜力与技术手段的优势，深入探索传统文化传播与神话叙事表达的新方式，创作出更多优秀的国产动画作品，推动中华文化在全球范围内的传播与发展。

# 侯孝贤电影研究-论侯孝贤不同时期电影创作的风格特征

罗云杉

(西北政法大学 陕西 西安 710122)

【摘要】侯孝贤导演是台湾新电影的先锋人物，亦是蜚声国际的当代电影大师，他的作品兼具文化反思与诗意表达，在城市现代化进程中关注个体感知和个体差异，将镜头聚焦于被冲击淹没的边缘人物，反映了通过电影艺术进行人文主义表达的一种方式。乡村与城市两种意识形态的对照、个体生命在历史变迁中所经历的阵痛始终是侯孝贤的作品所关注的文化母题。长镜头美学、抒情美学、时间与空间的节奏和对普通人的生活的关注、对社会现实的独特诠释与诗意化的叙事等成为侯孝贤电影的标签。本文从电影美学视域出发，以侯孝贤三个重要阶段的电影为研究对象，通过分析导演不同时期的拍摄和叙事风格，研究侯孝贤电影这样的艺术片具有的现实意义。

【关键词】影视美学；创作风格；拍摄技法；影视内容

# 一、创作内容

侯孝贤电影创作的能动性和独创性突出体现在其运用的别具一格的影像语言上，这种影像语言主要表现为“静观式”的长镜头、多种功能的空镜头与反线性叙事的剪辑方式的有机组合，通过这种独特影像语言的运用，侯孝贤既保证了影片的客观视角，又完成了主体意境的传达。从电影的纪实美学来看，侯孝贤的写实主义风格也暗合巴赞式的写实主义理论，大全景里长镜头的运用更好地展现了纪实题材的客观真实性，视线剪切等独特的拍摄手法又营造出诗意的、富有张力的意境，展现出东方美学的深厚意蕴，在平淡中见真章，饱含对人世沧桑的感慨。他的电影具有独特的个人风格和浓烈的个人色彩，在内容方面更注重人物感情的表达，重视生活细节带来的感悟。在影片构建的空间环境中，观众能从社会时代等大环境的变化中看到人物的成长，认识到人生的获得与失去。

# 二、影片风格

（一）第一阶段：“成长电影”里的美学

侯孝贤的早期电影喜爱讲他自己的故事，表明他具有一定的乡土情结，叙事风格是文艺片类型。

“成长四部曲”即从1983年开始依次上映的《风柜来的人》《冬冬的假期》《童年往事》和《恋恋风尘》四部作品。侯孝贤在一次讲座中谈道：“其实我们在童年时期以及成长的过程中，面对这个世界便已经有了一个眼光，是逃不掉的，不自觉的。其实在那时已经认识世界了，只是我们还说不清楚，所以这一段时间就会隐藏起来。”他此阶段的影片充斥着侯孝贤个人的青春记忆与怀旧情调，表现了其在台湾乡村成长的生命体验，带有个人自传色彩。这一时期，他对框式构图、长镜头、固定镜头与景深镜头等技法的运用逐渐成熟，为后来运用一拍到底的全景和展现含蓄的极简诗意风格奠定了基础。这一阶段的影片如《恋恋风尘》最具特色的是长镜头，找到一个位置，把镜头固定放在那儿一动不动，事情发生，这就突出了事件的真实性，减少了人物情感的主观干预。在导演的长镜头里，掌握主动权的是时间，导演没有在一定的时间里进行事件的表现，而是通过时间线的流逝自然形成故事。

《恋恋风尘》中，看似苦涩的青春初恋故事，实际在时间转换中以小见大地展现了城乡差异下不同个体的成长蜕变，透露出浓厚的台湾本土气息和对乡土的愁思。在农耕文明与现代工商业文明的变迁中,贯穿表现了时代变革与个人成长，主人公阿远的个体成长与当下社会也紧紧相连，表现了个体面对大环境时的无奈与顺从，如阿远寻找生计、不幸丢车、服兵役等，生命中的一系列聚散离别都不为自己所控，阿远和阿云离开家乡去城市打拼既是大时代背景下谋求生存不得已的选择，也是他们个人成长过程中的必经之路。在阿远离开家乡、遭受情感创伤并卧床痛哭之后，导演用一个长镜头里黎明破晓时的森林画面暗示阿远的悲痛如同火车穿过森林一样，个人的成长之痛在时代面前又能如何面对，一切都会随时间、随成长而过去。

这一阶段，他的影片在表达风格上注重写实，外加平淡如水的叙事风格，用大量的长镜头拍摄主人公行走在矿山小镇的画面，穿插的空镜头定格了青山、阴云、由红变绿的信号灯、无人等候的站台、散于风化作尘的初恋犹如娓娓道来。这部文艺青春电影横向上以城市和乡村为两个不同空间为对比，纵向上以阿远和父亲、阿公三代人面对生活的不同表现为线索。其中，火车是沟通城市与乡村两种文化空间的纽带，在城市和乡村两种空间里发挥着重要作用，火车的开动意味着两种文化氛围的转化与过渡。为了体现差异，导演在表现两个空间时所使用的镜头景别也不同，表现乡村更多使用远景镜头和全景镜头，如远处的山、静静的云，表现城市则通过拍摄凌乱的画室、拥挤的街头和狭窄空间里的人，城市的镜头里也把景深控制得很小，人物像被压缩在一个很局促的空间里，通过两个空间的对比使悠然的乡土文化在淡淡的音乐声中显得更加美好。

# （二）第二阶段：“历史主题电影”里的美学

导演这一阶段的作品聚焦真实的集体式历史回望，这一时期，侯孝贤导演开始关注历史、人类及文化等宏大主题。“台湾历史三部曲”中《悲情城市》和《好男好女》都蕴含着强烈的台湾历史文化特点和丰富的情感内涵，这一阶段的电影是对台湾过去和当下社会现状的记述，从文化、政治、经济方面展现台湾在大时代中的变化。作为“一名用影像叙事体进行表述的思想家”[2]，侯孝贤以独特的电影叙事语言和影像风格真实地记录了台湾的历史与现在，展现了台湾百年来的变迁，从而呈现在社会变迁中普通人的生活状态。这一阶段的代表作有《悲情城市》，其讲述了台湾近代史中“二·二八”事件大背景下林氏家族兄弟四人的遭遇和生活变化。在此之前，侯孝贤致力于从人物个体的视觉角度展现台湾乡土社会的工业化与城市化、现代化转型。对这部电影，有人批评说其对“二·二八”事件的展现比较模糊、克制，但在笔者看来，这除了受导演自身美学风格的影响外，更多的是因为导演在追求一种客观的叙事视角，和《聂隐娘》刻意藏锋掩鞘，只为营造主角疏离孤独的状态不同。侯孝贤不是20世纪50年代台湾社会事件的亲历者，他所做的不过是从一个旁观者的角度进行叙述，沉默往往是最有力的倾诉，很多时候，沉静淡然的空镜，固定不动的长镜头，有超越具有强烈情感冲突的镜头的冲击力。对于历史事件，人们对其的审视往往建立在某个特定阶级或群体的立场上，却时常忽视个体视角中个人在历史事件中变幻无常的命运更迭，而电影对关键事件的模糊化处理在某种程度上是对这种个体叙事的合理支撑。侯孝贤的影片总在叙述“家”的故事，而这个“家”总是渐渐式微，从个人小家到历史洪流中的国家，其展现了个人小家在近代史洪流中的渺小和无力，以传统的叙事方式透视社会与历史时代的变迁。法国电影理论家安德烈·巴赞在《摄影影像的本体论》中提出“摄影的美学特征在于它能揭示真实”[3]，在此视角下，电影艺术比任何艺术都更接近生活，所以，侯孝贤选择了具有个人风格的长镜头和固定机位进行叙事，聚焦日常生活中最重要的几个空间展现人物生活的状态，如林家正宅、照相馆。要想让镜头与空间融洽相处，就要处理好空间与人物的关系，用长焦距来展现电影空间要注意透视，强调镜头的“深”和“远”，反映历史的客观性和生命的状态。冯友兰先生曾将“禅”视为一种“静默的哲学”，在电影艺术中，“禅意”作为影像风格所呈现的审美意蕴在不同导演的创作中有不同的体现，可以是仙风道骨般的隐逸飘然，也可以是直面世俗人情的静观默看，侯孝贤导演的作品可视为禅意美学的典范之作，其影像风格的表达接近后者，他的电影以大量固定机位和镜头形成的长镜头段落表现真实的现实空间，还原二十世纪八九十年代的现实和生活原貌，给观众时间进行独立思考。摒弃大量特写与中景、近景镜头，而用远景、小全景来捕捉与展现有限的场景中人物的关系，门框、窗栏摄入镜头，形成绘画的意味，内涵层次非常丰富。

在叙事情节上，侯孝贤的电影不是用矛盾冲突来引导叙事情节的发展，而是用情感推动人物情绪的变化来侧面塑造影片的结构，通过对普通人尤其是下层民众生活的描写和刻画，以小见大地反映当下现象，聚焦社会问题，淡化传统的外在戏剧冲突，更加注重影片内在的张力，用情绪带动共鸣、推动故事情节发展。在第二阶段的历史性题材电影中，侯孝贤电影的两个关键特点是诗意化的含蓄表达风格与写实化的内容，他一改之前第一阶段惯用的叙事风格，回归心灵的深处，将浓烈的情感寄托于故事的时间和空间之中。相较于同时期的西方电影，侯孝贤的电影有横云断山的气魄，又如同泼墨的山水画，具有独特的中国传统美学意蕴。

（三）第三阶段“都市故事题材”里的美学侯孝贤这一时期的电影不再着眼于台湾乡村图景与历史，而是把视角转向现代都市青年。《南国再见，南国》和《千禧曼波》等都市题材表现的是人与人之间的疏离。这一时期的作品除了延续侯孝贤以往的电影风格之外，导演还用创意加持了配乐与光影设计，暖味多义的情调、动感十足的电子音乐也给观众展现了一个充满活力且不羁的侯孝贤，而音乐色彩背后让人无法忽视的画面展现了小人物的悲欢离合，讲了混混的故事，但没有打架、枪战和暴力，只有关于生活的描写，混混这一身份也仅仅代表他们是社会边缘的一员，也有梦想，对生活也感到彷徨无望，也有对家人的珍视，也需要努力才能换来生活的柴米油盐。导演没有拍摄人物的近景特写，大部分电影场景都停留在中景，画面里总是有两个以上的人，镜头就好像扫街般极具故事感。开场流动的长镜头把观众的目光聚焦在城市南部边界空间的状貌上，镜头跟随列车穿过乡镇两侧，摄影机作为媒介，成为大屏幕和观看者之间的隐形屏障，似乎把观众推向远方，并使视觉空间慢慢消失在屏幕深处，同样的火车镜头也出现在《恋恋风尘》里，此类流动的长镜头已成为其电影的标志性特征，导演用这种长焦摄影创造了一种视觉感官氛围，并使田园诗般的记忆成为象征。另外，在这一阶段，《南国再见，南国》的剪辑方式较之前有所突破，为我们提供了关于时空再现的一种新的思考方式。《南国再见，南国》中，侯孝贤导演数次运用纵深方向的镜头运动，带来了叙事之外的情绪高光。最典型的就是开头的铁轨后退镜头，中间那一组被调成红绿单色的主观镜头，虽具备叙事功能，但对人物情绪状态进行了外化展示，纵深方向的镜头运动更接近主观，代表对个体内心的聚焦，仿佛带领我们进入人物的内心，同时，它们也在叙事的间隙让观者得以进行短暂的休憩。通过在两种运镜方式间有节奏地切换，侯孝贤成功制造了一种“入梦，出梦”的体验。这组视觉上的对比实际也承载了《南国再见，南国》的主题，横移段落代表主人公们停滞黏着的现实，而纵深镜头尤其是正推与倒拉镜头，代表着人物前进和告别的运动趋势，而影片最后证明了这种运动的结果是虚无。《南国再见，南国》的另一个不同之

处在于人工美学的加入，很多超现实主义打光的段落结合酷烈的现代音乐，都让人想到王家卫《堕落天使》式的都市美学。这种美学和导演本身的乡土美学碰撞出了另一种火花，丰富了电影在视觉上的层次。

# 三、结语

侯孝贤的电影是诗化风格的电影，即诗意电影，是中国哲学、中国宗教、中国诗词影响下的美学感悟的自然流露，讲究中国绘画般的留白与布局，他将地平线置于画面边缘，这样的空灵构图可以实现艺术创作中写实与写意的结合。叙事单元中，长镜头的运用通过反对变焦，大大减少了主观干预，更让人感受到时空的完整与统一。镜头是侯孝贤电影的一个鲜明特征，在他的电影中，很多戏都是全景一拍到底，观众在观看侯孝贤的电影时，总是和剧中人物保持一定的距离，即使在想看清人物表情动作的时候，导演仍然坚持不懈地运用这种镜头，产生了一种极简含蓄的风格，淡化了叙事，这是其电影诗意风格形成的重要基础。其电影中大多数场景运用固定长镜头，也对应着时局的压抑、人物生存的被动，展现无奈和拘禁感，可以保持画面内容与人物情绪心境相互关系的稳定，使人、景、物融为一体，以便让观众产生共鸣，用特有的东方审美去表达和建构他的乌托邦，重塑传统文化中天人合一的境界。
